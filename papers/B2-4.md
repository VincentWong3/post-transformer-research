# Focal Self-attention for Local-Global Interactions in Vision Transformers（2021）

**论文：** [arXiv](https://arxiv.org/abs/2107.00641) · Yang et al. · 2021

---

**核心思路：粗细粒度结合的注意力——近处全分辨率、远处池化降采样，类似人眼的 Fovea 机制。**

## Focal Attention

注意力不是所有 token 同等处理，而是：
- **Fine level**：查询 token 的局部邻域（近距离）
- **Coarse level**：全局视角但降采样后（远距离）

```python
# Fine level: 局部 3×3 窗口 attention
Q @ K_fine.T   # K_fine 来自邻域 patches
# Coarse level: 全局 attention，但 K/V 经过 AvgPool 降维
Q @ K_coarse.T # K_coarse 来自 2× or 4× 降维后的全局图
```

## 结果

Focal-T（29M）在 ImageNet 上 82.0%，比 Swin-T（81.3%）高 0.7 个点。但实际延迟略高于 Swin。Focal 的关键洞察是：**不是所有空间位置都需要全精度的 attention——远处的物体可以粗粒度处理。**
