# End-to-End Object Detection with Transformers（2020）

**论文：** [arXiv](https://arxiv.org/abs/2005.12872) · Carion et al. · 2020（DETR）

---

**核心思路：用 Transformer encoder-decoder 替代目标检测中的 proposal / anchor 流水线，将检测视为一个 set prediction 问题。**

## 关键创新

1. **Set-based loss**：用匈牙利算法（bipartite matching）匹配预测框和 GT 框——不需要 NMS 后处理
2. **Object queries**：100 个可学习的 query 向量，每个对应一个潜在目标。经过 decoder 后输出（class, box）
3. **Transformer encoder-decoder**：encoder 对 CNN 特征加位置编码后做自注意力；decoder 用 object queries 交叉注意力到 encoder 输出

## 结果

DETR 在 COCO 上 42.0 AP（ResNet-50），与 Faster R-CNN 相当。但：
- **优点**：不需要 anchor 设计、不需要 NMS、架构简洁
- **缺点**：训练收敛慢（500 epoch vs 100 epoch）、小目标检测差

DETR 开创了基于 Transformer 的端到端检测范式。后续工作（Deformable DETR、DINO、DAB-DETR）逐步解决了 DETR 的收敛速度和小目标问题。
