# Momentum Contrast for Unsupervised Visual Representation Learning（2020）

**论文：** [arXiv](https://arxiv.org/abs/1911.05722) · He et al. · 2020（MoCo）

---

**核心思路：用 momentum encoder 维护一个大的负样本队列，解耦 batch size 和负样本数。**

## MoCo v1

MoCo 的三个组件：
1. **Query encoder**（正在训练的网络）
2. **Key encoder**（query encoder 的 momentum 版本：$\theta_k \leftarrow m\theta_k + (1-m)\theta_q$）
3. **Queue**（FIFO 队列，存储 key encoder 的历史输出，作为负样本）

关键是 momentum encoder 和 queue 解耦了 batch size 和负样本数——可以用小 batch（256）维护 65,536 个负样本。

## MoCo v2

在 MoCo v1 的基础上加入 SimCLR 的两个改进：
1. **MLP 投影头**（替换原始的全连接线性头）
2. **更强的数据增强**（ColorJitter + GaussianBlur）

MoCo v2 在 200 epoch 下达到 71.1% linear probe (ResNet-50)，与 SimCLR 的 71.4% 打平但 batch size 只有 SimCLR 的 1/16（256 vs 4096）。

## MoCo v3

用 ViT 替换 ResNet 作为编码器，发现 ViT 在对比学习中训练不稳定（loss spikes）。解决方案：冻结 patch projection layer 的第一层。
