## 一、先搞清楚坑在哪

自监督学习的核心目标是让模型在没有标签的情况下学到有用的视觉表征。2020 年前后，对比学习（contrastive learning）是这个领域的主流范式：让同一张图的不同增强视图（positive pairs）在特征空间里靠近，让不同图的视图（negative pairs）互相远离。

但对比学习有个绕不开的痛点——**需要负样本**。MoCo [He et al., 2020] 用动量编码器维护大队列来提供负样本，SimCLR [Chen et al., 2020] 用大 batch size 来塞够负样本。这些设计增加了工程复杂度，也让人忍不住想问：**能不能彻底扔掉负样本？**

直觉上，只用正样本做"拉近"会导致 collapse——模型把所有图都映射到同一个点，损失函数就废了。怎么防止 collapse，成了自监督学习的核心难题。

## 二、现有方法的真正问题

SimSiam 之前，已经有几种方法在尝试解决 collapse 问题，但各有一身"包袱"：

- **MoCo**：动量编码器 + 负样本队列，工程复杂，需要维护两个编码器。
- **SimCLR**：大 batch（4096），对显存要求极高，训练不稳定。
- **BYOL [Grill et al., 2020]**：第一个证明只用正样本也能学，但它依赖**动量编码器**来防止 collapse。动量为什么有效？理论解释很模糊，更像一个工程 trick。
- **SwAV [Caron et al., 2020]**：用在线聚类代替负样本，需要 Sinkhorn-Knopp 算法，流程复杂。

这些方法核心区别在于"怎么防 collapse"。但一个更根本的问题是：**这些花哨的机制是不是真的必须？** 有没有可能，一个极其简单的结构就能搞定？

## 三、SimSiam 的核心思路

SimSiam 的答案简单到让人怀疑：**什么都不用加**——没有负样本、没有动量编码器、没有大 batch、没有聚类。只需要一个 Siamese 网络（共享权重的双分支网络，两个分支使用同一个编码器处理不同的增强视图）+ 预测器 + **stop-gradient（停止梯度传播）**，就能防止 collapse。

具体来说：对同一张图做两次随机增强得到 $x_1$、$x_2$，两个视图通过同一个编码器 $f$（共享权重），一个视图的编码结果再过预测器 $h$，另一个视图直接输出，两者算负余弦相似度损失。关键操作是：**梯度只更新预测器那一侧，另一侧 stop-gradient**。

就这么简单。没有动量，没有队列，没有聚类。

```
输入图片 x
    │
    ├── 增强 T1 ──→ x1 ──→ 编码器 f (共享) ──→ z1 ──→ 预测器 h ──→ p1
    │
    └── 增强 T2 ──→ x2 ──→ 编码器 f (共享) ──→ z2 ──→ stopgrad ──→ sg(z2)
                                                                  │
                                              损失: -cosine(p1, sg(z2))
```

SimSiam 可以被看作是"去掉动量编码器的 BYOL"、"去掉负样本的 SimCLR"、"去掉在线聚类的 SwAV"——它跟每个方法都只差一步，但居然也能 work。

下面我们来具体拆解它的网络结构和 stop-gradient 的工作原理。

## 四、方法细节

### 4.1 网络结构

- **编码器 $f$**：ResNet-50（去掉最后的全连接层）+ 3 层投影 MLP（2048-d，每层带 Batch Normalization（BN），输出层无 ReLU）。
- **预测器 $h$**：2 层 MLP，结构为 FC(2048) → BN → ReLU → FC(512) → BN → FC(2048)。注意是 bottleneck 结构（2048→512→2048），输出层没有 BN。
- 两个分支**完全共享编码器权重**，不像 BYOL 那样用动量更新。

### 4.2 为什么 stop-gradient 有效？

这是 SimSiam 最核心的发现。论文给出了一个**隐式 EM 算法**的理论框架。这里 $F_\theta$ 就是前面的编码器 $f$，$\theta$ 是编码器参数。作者将问题建模为：每张图片 $x_i$ 应该有一个稳定的表征，这个表征对数据增强是不变的。于是可以把优化目标写成：

$$\min_{\theta, \eta} \sum_i \mathbb{E}_{T} \left\lVert F_\theta(T(x_i)) - \eta_{x_i} \right\rVert_2^2$$

其中 $\theta$ 是编码器参数，$\eta_{x_i}$ 是每张图的"表征"（类似聚类中心）。这个目标可以交替优化：

- **E-step**：固定 $\theta$，求解 $\eta_{x_i} = \mathbb{E}_T[F_\theta(T(x_i))]$——即取多个增强视图的平均。
- **M-step**：固定 $\eta$，用 SGD 优化 $\theta$——即最小化 $\|F_\theta(T(x_i)) - \eta_{x_i}\|^2$。

SimSiam 的 stop-gradient 恰好模拟了**一步 E-step + 一步 M-step 的交替**。当我们在 $x_1$ 上做预测、$x_2$ 上 stop-gradient 时，$x_2$ 的输出相当于伪标签 $\eta$，$x_1$ 的预测器去拟合它。下次迭代伪标签更新了，预测器再去追。这个交替过程天然防止 collapse，因为 $\eta$ 是固定的，不会跟着预测器一起坍缩。

### 4.3 关键消融

论文通过大量实验证明了三个关键发现：

1. **stop-gradient 是防止 collapse 的唯一必要条件**——没有它，模型立即坍缩到随机猜测（0.1% top-1）。
2. **预测器是提升性能的必要组件**——去掉它性能从 67.4% 掉到 55.0%，但没有 collapse。
3. **动量编码器可以被 stop-gradient 完全替代**——动量版本和 stop-gradient 版本性能相同。

## 五、公式详解

### 损失函数

SimSiam 使用负余弦相似度作为损失核心，逐视图对称计算。定义如下：

$$D(p_1, z_2) = -\frac{p_1}{\lVert p_1 \rVert_2} \cdot \frac{z_2}{\lVert z_2 \rVert_2}$$

其中 $p_1 = h(f(x_1))$，$z_2 = f(x_2)$ 且施加 stop-gradient。两个向量都 L2 归一化到单位球面。

**直觉**：这就是两个单位向量之间的**负余弦相似度**。方向一致时，余弦相似度为 1，损失 = -1；方向相反时损失 = 1。最小化损失 = 让两个向量指向同一方向。

**对称损失**：

$$L = \frac{1}{2} D(p_1, \text{sg}(z_2)) + \frac{1}{2} D(p_2, \text{sg}(z_1))$$

其中 $\text{sg}$ 表示 stop-gradient。两个视图都过一遍，但每次梯度只更新预测器那一侧。最小值是 $-1$。

## 七、SimSiam 在领域中的位置

### 前驱工作

SimSiam 跟每个主流方法的关系都是**"去掉一个核心组件"**：去掉 BYOL 的动量编码器、去掉 SimCLR 的负样本、去掉 SwAV 的在线聚类。它证明所有这些组件对防止 collapse 都不是必需的，**stop-gradient 才是本质**。

### 后续影响

SimSiam 对后续自监督学习产生了深远影响：

1. **DINO [Caron et al., 2021]**：将 stop-gradient 扩展到了 Vision Transformer 的自蒸馏训练中。
2. **MoCo v3 [Chen et al., 2021]**：发现 stop-gradient 对 ViT 的自监督训练至关重要。
3. **标准基线**：SimSiam 成为自监督学习最简洁的 baseline，是研究新问题时 "先试这个" 的默认选择。

## 八、局限

1. **对数据增强的依赖**：SimSiam 的性能高度依赖于数据增强策略（随机裁剪、颜色抖动、高斯模糊）。增强不够多样时，模型容易学到捷径。
2. **对小 batch 的 BN 敏感**：虽然对 batch size 整体不敏感，但 batch size < 64 时 BN 统计量不稳定，论文没有深入探索。
3. **理论解释是 post-hoc 的**：EM 假设是在观察 stop-gradient 有效之后提出的，而不是事先设计推导出来的。为什么 stop-gradient 有效，仍然缺少严格的数学证明。
4. **ViT 兼容性受限**：SimSiam 原本在 ResNet 上验证。后续发现它对 ViT 的训练会不稳定，需要额外技巧（如冻结 patch embedding 层），说明 stop-gradient 不是所有架构的通用解。

## 九、小结

SimSiam 的核心贡献是：**揭示了 stop-gradient 是防止自监督学习 collapse 的关键机制**，而此前被认为必不可少的动量编码器、负样本、聚类等都是多余的。

这个工作的 broader impact 是：把自监督学习从"精心设计各种防 collapse 机制"的复杂工程问题，简化成了"只需要 stop-gradient"的优雅问题。它提醒了领域——有时候最简单的答案就是最好的答案。
