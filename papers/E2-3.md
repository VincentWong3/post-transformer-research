# FNet: Mixing Tokens with Fourier Transforms（2021）

**论文：** [arXiv](https://arxiv.org/abs/2105.03824) · Lee-Thorp et al. · 2021

用 2D FFT 替代 attention——在空间维度和 channel 维度做 Fourier 变换后在频域乘可学习的权重。$O(N \log N)$ 复杂度，比 attention $O(N^2)$ 高效。在 GLUE 上达到 BERT 的 92-97% 性能——说明**频域变换也可以有效地混合 token 信息**。

# Efficient Token Mixing for Transformers via Adaptive Fourier Neural Operators（2022）

**论文：** [arXiv](https://arxiv.org/abs/2111.13587) · Guibas et al. · 2022（AFNO）

基于 Fourier Neural Operator 的 token mixing，用**自适应频域滤波**替代 attention。适合高分辨率输入。在气象预报和医学图像任务上表现好。

# Fourier Neural Operator for Parametric Partial Differential Equations（2020）

**论文：** [arXiv](https://arxiv.org/abs/2010.08895) · Li et al. · 2020（FNO）

在 PDE 求解任务上用 Fourier 层做 token mixing。FNO 在频域学习核函数，不需要知道 PDE 的具体形式，学习 PDE 解算子。在 Navier-Stokes 等复杂 PDE 上达到 SOTA。
