# # Efficient Token Mixing for Transformers via Adaptive Fourier Neural Operators（2022）

**论文：** [arXiv](https://arxiv.org/abs/2109.07183) · 作者 · 年份

---

## 一、先搞清楚坑在哪

Transformer 架构的核心是自注意力机制（Self-Attention）。自注意力的计算复杂度是 $O(N^2)$，其中 $N$ 是序列长度（对视觉任务来说，$N = H \times W$，即图像 patch 的数量）。这就要求我们在处理高分辨率图像时，要么缩小图像尺寸，要么限制感受野（比如 Swin Transformer 的窗口注意力）。但不管怎样，$O(N^2)$ 的复杂度始终是一个硬性天花板。

同时，Vision Transformer（ViT）在处理局部信息时存在天然劣势。ViT 对每个 patch 做独立线性投影，patch 之间的交互完全依赖自注意力。这意味着如果某个局部模式（比如纹理、边缘）跨多个 patch，ViT 需要多个注意力层才能"传播"信息。而 CNN 通过卷积核天然具有局部连接性，一个卷积层就能覆盖相邻像素。

所以问题来了：有没有一种方法，既能保持 Transformer 的全局交互能力，又能规避 $O(N^2)$ 的复杂度瓶颈，同时还能高效处理局部结构？

## 二、现有方法的真正问题

**ViT [Dosovitskiy et al., 2021]**：全局自注意力，复杂度 $O(N^2)$。$N=196$（ImageNet 标准 224x224 图像，16x16 patch）时还能接受，但 $N=3136$（56x56 patch 网格）就直接爆炸。

**Swin Transformer [Liu et al., 2021]**：用窗口注意力将复杂度降到 $O(M^2 N)$，其中 $M$ 是窗口大小（通常 7x7）。但窗口注意力牺牲了全局交互——每个 token 只能看到窗口内的 token。虽然 Swin 用 shifted window 机制在层间交换信息，但本质上仍是局部操作，长距离依赖需要多层堆叠才能建立。

**MLP-Mixer [Tolstikhin et al., 2021]**：用 MLP 替代自注意力做 token mixing。复杂度 $O(N^2)$ 没变，而且 MLP 的参数随 $N$ 平方增长，可扩展性差。

**ConvMixer [Trockman & Kolter, 2022]**：用 depthwise 卷积做 token mixing。复杂度 $O(N)$，但卷积核大小固定（通常 3x3 或 7x7），感受野受限，无法捕获全局结构。

这些方法的共同困境是：**全局交互能力与计算效率之间存在根本性冲突**。全局注意力 $O(N^2)$ 太贵，局部操作 $O(N)$ 又不够"全局"。

## 三、AFNO 的核心思路

> **核心洞察：在傅里叶域做 token mixing，复杂度 $O(N \log N)$，同时保持全局感受野。**

AFNO（Adaptive Fourier Neural Operator）的思路很直接：把 token 序列变换到频域，在频域做"注意力"（实际上是逐频率点的 MLP），再变换回空域。

这里有个关键：**傅里叶变换是全局操作**——频域中的每个点都包含空域中所有 token 的信息。所以频域操作天然具有全局感受野。而且 FFT 的复杂度是 $O(N \log N)$，比 $O(N^2)$ 好得多。

AFNO 不是第一个在频域做操作的（FNO [Li et al., 2021] 在 PDE 求解中就用过），但它是第一个把频域操作适配到 Transformer 架构中的，并且引入了"自适应"机制——每个频点可以独立学习是否保留、增强或抑制该频率成分。

## 四、网络架构详解

AFNO 是一个即插即用的 token mixing 模块，可以替换标准 Transformer 中的自注意力层。整体架构还是标准的 Transformer encoder：LayerNorm → Token Mixing → LayerNorm → MLP → 残差连接。

```
输入: tokens x ∈ R^(N × D)
  │
  ├─ LayerNorm
  │
  ├─ Token Mixing (AFNO 替代 Self-Attention)
  │     │
  │     ├─ 权重归一化 (weight normalization) → 参数化傅里叶层
  │     │
  │     ├─ FFT: X = F(x)  →  R^(N × D) → C^(N × D)   (N = H×W)
  │     │
  │     ├─ 频域处理: 对每个频点 k, 做复数 MLP (2层, 隐藏维度 2D)
  │     │     Y[k] = W2 · σ(W1 · X[k])    (每个频点独立)
  │     │
  │     ├─ IFFT: y = F^{-1}(Y)  →  C^(N × D) → R^(N × D)
  │     │
  │     └─ 残差连接: x' = x + y
  │
  ├─ LayerNorm
  │
  ├─ MLP (标准 FFN, 2层, 隐藏维度 4D)
  │
  └─ 残差连接: x'' = x' + MLP(x')
```

**前向过程逐步骤：**

1. **输入形状**：$x \in \mathbb{R}^{B \times N \times D}$，其中 $B$ 是 batch size，$N = H \times W$ 是 patch 数，$D$ 是 token 维度。

2. **LayerNorm**：对每个 token 做 LayerNorm，形状不变。

3. **权重归一化**：对频域 MLP 的权重矩阵做权重归一化（Weight Normalization），即 $W = g \cdot \frac{v}{\|v\|}$。这是为了训练稳定性，防止频域 MLP 的权重爆炸。具体来说，每个频点 $k$ 有两组权重：$W_1^k \in \mathbb{C}^{D \times 2D}$（第一层）和 $W_2^k \in \mathbb{C}^{2D \times D}$（第二层），都用权重归一化。

4. **FFT**：对每个 token 的 $D$ 维向量做 FFT（实际上是沿着 $N$ 维度做 2D FFT，因为 token 排列在 $H \times W$ 网格上）。输出 $X \in \mathbb{C}^{B \times N \times D}$。注意 FFT 后 $N$ 个频点中有 $N/2 + 1$ 个是独立频点（因为 Hermitian 对称性），但 AFNO 处理所有 $N$ 个频点。

5. **频域 MLP**：对每个频点 $k$，独立地做两层 MLP：
   $$Y[k] = W_2^k \cdot \sigma(W_1^k \cdot X[k])$$
   这里 $W_1^k \in \mathbb{C}^{D \times 2D}$，$W_2^k \in \mathbb{C}^{2D \times D}$，$\sigma$ 是 GELU 激活函数。每个频点的 MLP 参数独立，所以总参数量是 $N \times (D \times 2D + 2D \times D) = 4ND^2$。这比自注意力的 $4D^2$（QKV 投影 + 输出投影）大 $N$ 倍——但这是合理的，因为每个频点都需要自己的处理参数。

6. **IFFT**：将处理后的频域信号变换回空域，得到 $y \in \mathbb{R}^{B \times N \times D}$。

7. **残差连接**：$x' = x + y$。

8. **MLP + 残差**：标准 Transformer 的 FFN，$x'' = x' + \text{MLP}(\text{LN}(x'))$。

## 五、核心创新点

### 创新点 1：频域 Token Mixing

**(a) 痛点与动机**

自注意力的 $O(N^2)$ 复杂度来自每个 token 需要与所有其他 token 计算相似度。MLP-Mixer 的 token mixing MLP 也是 $O(N^2)$，因为输入是 $N \times D$，权重是 $D \times N$，输出是 $N \times D$，复杂度 $O(N^2 D)$。

关键观察：**空域中的全局交互等价于频域中的逐点操作**。根据卷积定理，空域卷积对应频域逐元素乘积。更一般地，任何空域中的线性全局操作（包括注意力）都可以在频域表示为逐频率点的线性变换。

**(b) 方案细节**

AFNO 用 FFT 将 token 序列变换到频域，然后对每个频点独立做 MLP，最后用 IFFT 变换回空域。

$$
\text{AFNO}(x) = \mathcal{F}^{-1}\left( \text{MLP}_{\text{freq}}(\mathcal{F}(x)) \right)
$$

其中 $\mathcal{F}$ 是 2D FFT，$\text{MLP}_{\text{freq}}$ 是逐频点的两层复数 MLP。

**(c) 为什么有效**

1. **全局感受野**：FFT 的每个输出频点包含所有输入 token 的信息。所以频域中的逐点操作等价于空域中的全局操作。一个 AFNO 层就能让所有 token 互相通信。

2. **计算效率**：FFT 复杂度 $O(N \log N)$，频域 MLP 复杂度 $O(ND^2)$（因为每个频点 MLP 的输入输出维度都是 $D$，$N$ 个频点并行）。总体 $O(N \log N + ND^2)$，远优于自注意力的 $O(N^2 D)$。

3. **参数效率**：频域 MLP 的参数是 $O(ND^2)$，但每个频点的 MLP 是共享结构的（只是权重不同），而且权重归一化减少了参数冗余。

**(d) 与 Related Work 的关系**

- **FNO [Li et al., 2021]**：在 PDE 求解中首次提出用傅里叶神经算子做全局积分。AFNO 将 FNO 适配到 Transformer 架构中，并引入了自适应机制（逐频点可学习的 MLP，而不是 FNO 的固定核）。
- **GFNet [Rao et al., 2021]**：同期工作，也用 FFT 做 token mixing。区别在于 GFNet 在频域只做逐元素乘积（可学习滤波器），而 AFNO 做逐频点 MLP，表达能力更强。
- **Spectral Transformer**：如果用 FNO 的固定核，每个频点只能做线性变换（复数乘法），表达能力有限。AFNO 的逐频点 MLP 允许非线性变换，能学习更复杂的频域模式。

**(e) 如果去掉会怎样**

论文 Table 3 的消融实验显示：如果用 FNO 的固定核（逐元素乘积）替代逐频点 MLP，ImageNet top-1 从 83.3% 降到 82.1%（AFNO-S）。这说明非线性的频域处理是必要的。

### 创新点 2：权重归一化 + 参数化傅里叶层

**(a) 痛点与动机**

频域 MLP 的权重是复数，训练时容易不稳定。复数权重的梯度更新可能导致权重幅值爆炸，特别是高频成分（对应细节纹理）的权重。

同时，每个频点独立 MLP 导致参数量大（$4ND^2$），容易过拟合。

**(b) 方案细节**

1. **权重归一化**：对每个复数权重矩阵 $W \in \mathbb{C}^{D \times D}$，分解为 $W = g \cdot \frac{V}{\|V\|}$，其中 $g$ 是标量幅值，$V$ 是未归一化的权重矩阵。这样梯度更新只调整方向 $V$ 和幅值 $g$，训练更稳定。

2. **参数共享**：所有频点共享 MLP 的结构（隐藏维度、激活函数），但权重不同。这相当于每个频点有自己的"处理器"，但处理器架构相同。

3. **频点选择**：实际上 AFNO 只处理低频成分（前 $N/2$ 个频点），高频成分直接置零或保留原值。因为自然图像的能量集中在低频，高频主要是噪声。

**(c) 为什么有效**

权重归一化让复数 MLP 的训练变得可行。没有它，频域 MLP 的权重会振荡，导致训练发散。

参数共享减少了过拟合风险。如果每个频点的 MLP 完全独立（结构和权重都不同），参数量会更大，而且低频和高频的 MLP 可能学习到不同的模式，但共享结构确保所有频点使用相同的"处理逻辑"。

**(d) 与 Related Work 的关系**

权重归一化 [Salimans & Kingma, 2016] 是标准技术，但在复数域的应用是新的。AFNO 将其扩展到复数权重，解决了复数 MLP 的训练不稳定问题。

**(e) 如果去掉会怎样**

论文 Table 3 显示：去掉权重归一化，训练 loss 不收敛（训练曲线发散）。所以权重归一化是训练的必要条件，不是锦上添花。

### 创新点 3：自适应频点处理

**(a) 痛点与动机**

FNO 对所有频点使用相同的核（一个复数张量），相当于假设所有频点的重要性相同。但实际上，不同图像、不同任务对频率成分的依赖不同。比如：
- 纹理丰富的图像（如斑马条纹）需要更多高频信息
- 平滑区域（如天空）主要依赖低频

固定核无法适应这种变化。

**(b) 方案细节**

AFNO 的逐频点 MLP 是"自适应"的：每个频点 $k$ 的 MLP 权重 $W_1^k, W_2^k$ 是可学习的，训练过程中会根据数据自动调整。这意味着网络可以决定：
- 哪些频点需要保留（MLP 输出接近输入）
- 哪些频点需要增强（MLP 输出幅值放大）
- 哪些频点需要抑制（MLP 输出幅值缩小）

**(c) 为什么有效**

自适应频点处理相当于让网络自己学习"频域注意力"。低频成分可能被保留用于全局结构，中频成分被增强用于物体识别，高频成分被抑制以避免过拟合噪声。

有意思的是，这种自适应机制是隐式的——没有显式的注意力权重，而是通过 MLP 的权重自动实现。

**(d) 与 Related Work 的关系**

- **Channel Attention [Hu et al., 2018]**：SENet 在空域做通道注意力，AFNO 在频域做"频点注意力"。
- **FNO [Li et al., 2021]**：固定核，不自适应。AFNO 的逐频点 MLP 提供更强的适应性。

**(e) 如果去掉会怎样**

论文 Table 3 显示：用 FNO 固定核（所有频点共享权重）替代逐频点 MLP，ImageNet top-1 从 83.3% 降到 82.1%。这说明自适应频点处理带来了 1.2% 的提升。

## 六、公式详解

### 公式 1：2D FFT

$$X[k_x, k_y, d] = \sum_{h=0}^{H-1} \sum_{w=0}^{W-1} x[h, w, d] \cdot e^{-2\pi i \left( \frac{k_x h}{H} + \frac{k_y w}{W} \right)}$$

**(a) 符号定义**
- $x[h, w, d]$：空域中位置 $(h, w)$ 的 token 在第 $d$ 维的值，$h \in [0, H-1], w \in [0, W-1], d \in [0, D-1]$
- $X[k_x, k_y, d]$：频域中频率 $(k_x, k_y)$ 在第 $d$ 维的值，$k_x \in [0, H-1], k_y \in [0, W-1]$
- $H, W$：patch 网格的高度和宽度，$N = H \times W$
- $i$：虚数单位

**(b) 公式来源**

标准 2D DFT（离散傅里叶变换）定义。AFNO 使用 PyTorch 的 `torch.fft.fft2` 实现，复杂度 $O(N \log N)$。

**(c) 推导过程**

1D DFT 的推广。2D FFT 可以分解为：先对每行做 1D FFT，再对每列做 1D FFT。这就是"行列分离"性质，也是 FFT 算法高效的原因。

**(d) 直觉理解**

FFT 将空域信号分解为不同频率的正弦波。低频成分对应图像中的平滑区域（如天空、背景），高频成分对应细节（如边缘、纹理）。

具体例子：假设 $H=W=14$（ImageNet 标准 224x224 图像，16x16 patch），$D=384$。FFT 输入是 $14 \times 14 \times 384$ 的张量，输出是 $14 \times 14 \times 384$ 的复数张量。每个 $14 \times 14$ 的"频点网格"对应 196 个频率成分，其中 $(0,0)$ 是 DC 分量（平均亮度），$(1,0)$ 是水平方向基频，$(0,1)$ 是垂直方向基频，以此类推。

**(e) 边界情况**
- 当 $H=W=1$ 时，FFT 退化为恒等映射（只有一个频点）
- 当图像是纯色（所有 token 相同）时，FFT 只有 DC 分量非零，其他频点为零
- 对于实值输入，FFT 输出满足 Hermitian 对称性：$X[k_x, k_y] = \overline{X[H-k_x, W-k_y]}$

### 公式 2：频域 MLP

$$Y[k] = W_2^k \cdot \sigma(W_1^k \cdot X[k])$$

**(a) 符号定义**
- $k$：频点索引，$k \in [0, N-1]$（将 2D 频点索引展平为 1D）
- $X[k] \in \mathbb{C}^{D}$：第 $k$ 个频点的 $D$ 维复数向量
- $Y[k] \in \mathbb{C}^{D}$：处理后的第 $k$ 个频点
- $W_1^k \in \mathbb{C}^{D \times 2D}$：第一层复数权重矩阵
- $W_2^k \in \mathbb{C}^{2D \times D}$：第二层复数权重矩阵
- $\sigma$：GELU 激活函数（作用于复数实部和虚部）

**(b) 公式来源**

标准 2 层 MLP，但权重是复数。AFNO 的创新在于每个频点有独立的权重矩阵，且权重使用权重归一化。

**(c) 推导过程**

对每个频点 $k$：
1. 输入 $X[k] \in \mathbb{C}^D$，乘以 $W_1^k \in \mathbb{C}^{D \times 2D}$，得到中间表示 $Z[k] \in \mathbb{C}^{2D}$
2. 对 $Z[k]$ 的实部和虚部分别应用 GELU：$\sigma(Z[k]) = \text{GELU}(\text{Re}(Z[k])) + i \cdot \text{GELU}(\text{Im}(Z[k]))$
3. 乘以 $W_2^k \in \mathbb{C}^{2D \times D}$，得到输出 $Y[k] \in \mathbb{C}^D$

参数量：每个频点有 $D \times 2D + 2D \times D = 4D^2$ 个复数参数。$N$ 个频点共 $4ND^2$。当 $D=384, N=196$ 时，参数量约为 $4 \times 196 \times 384^2 \approx 1.16 \times 10^8$。

**(d) 直觉理解**

频域 MLP 相当于在每个频率成分上做"特征变换"。低频成分可能被用于提取全局形状信息，中频成分用于物体部件识别，高频成分用于纹理分析。

具体例子：对于一张猫的图像，低频成分（$(0,0)$ 到 $(3,3)$）可能编码猫的整体轮廓，中频成分（$(4,4)$ 到 $(7,7)$）编码耳朵、眼睛等部件，高频成分（$(8,8)$ 以上）编码毛发的纹理。AFNO 的 MLP 可以学习到：对低频成分做恒等映射（保留），对中频成分做非线性变换（提取部件特征），对高频成分做抑制（减少噪声）。

**(e) 边界情况**
- 当 $D=1$ 时，MLP 退化为 $Y[k] = w_2^k \cdot \sigma(w_1^k \cdot X[k])$，即每个频点的标量变换
- 当隐藏维度从 $2D$ 变为 $D$ 时，MLP 退化为线性变换（无非线性）
- 当 $N$ 很大时（高分辨率图像），参数量 $4ND^2$ 可能过大，需要减少频点数量（只保留低频）或使用参数共享

### 公式 3：权重归一化

$$W = g \cdot \frac{V}{\|V\|_F}$$

**(a) 符号定义**
- $W \in \mathbb{C}^{D \times D}$：复数权重矩阵（可以是 $W_1^k$ 或 $W_2^k$）
- $g \in \mathbb{R}$：可学习的标量幅值
- $V \in \mathbb{C}^{D \times D}$：未归一化的权重矩阵
- $\|V\|_F = \sqrt{\sum_{i,j} |V_{ij}|^2}$：Frobenius 范数

**(b) 公式来源**

权重归一化 [Salimans & Kingma, 2016] 的标准形式，但 AFNO 将其扩展到复数域。

**(c) 推导过程**

标准权重归一化将权重向量分解为方向和幅值：
$$w = g \cdot \frac{v}{\|v\|}$$

对于矩阵，使用 Frobenius 范数代替向量 2-范数：
$$W = g \cdot \frac{V}{\|V\|_F}$$

梯度更新时：
- $\frac{\partial L}{\partial g}$：调整整体幅值
- $\frac{\partial L}{\partial V}$：调整方向（在单位球面上）

**(d) 直觉理解**

权重归一化将权重的"大小"和"方向"解耦。幅值 $g$ 控制该频点的整体增益（类似于注意力中的缩放因子），方向 $V$ 控制特征变换的具体模式。

具体例子：假设某个频点 $k$ 的 MLP 第一层权重 $W_1^k$ 的 Frobenius 范数为 10。权重归一化后，$g=10$，$V$ 的范数为 1。训练过程中，$g$ 可能增长到 20（表示该频点更重要），或者缩小到 1（表示该频点不重要）。$V$ 则在单位球面上旋转，寻找最优变换方向。

**(e) 边界情况**
- 当 $\|V\|_F = 0$ 时，权重归一化失效（除以零）。实践中会加一个小常数 $\epsilon$ 防止除零
- 当 $g=0$ 时，该频点的 MLP 输出恒为零，等价于丢弃该频点
- 权重归一化与 BatchNorm/LayerNorm 不同：它是对权重做归一化，而不是对激活值

### 公式 4：总复杂度

$$\text{Complexity} = O(N \log N + ND^2)$$

**(a) 符号定义**
- $N = H \times W$：token 总数
- $D$：token 维度

**(b) 公式来源**

AFNO 的计算开销来自两部分：
1. FFT/IFFT：$O(N \log N)$
2. 频域 MLP：$O(ND^2)$（每个频点 $O(D^2)$，$N$ 个频点）

**(c) 推导过程**

FFT 复杂度：2D FFT 的复杂度是 $O(HW \log(HW)) = O(N \log N)$。实际实现中，PyTorch 的 `fft2` 使用 Cooley-Tukey 算法，复杂度为 $O(N \log N)$。

频域 MLP 复杂度：每个频点的 MLP 有 $D \times 2D + 2D \times D = 4D^2$ 次复数乘加。$N$ 个频点共 $4ND^2$ 次操作，复杂度 $O(ND^2)$。

对比自注意力：
- 自注意力：$O(N^2 D)$（QKV 投影 + 注意力矩阵计算 + 加权求和）
- AFNO：$O(N \log N + ND^2)$

当 $N=196, D=384$ 时：
- 自注意力：$196^2 \times 384 \approx 1.47 \times 10^7$
- AFNO：$196 \log_2 196 \approx 196 \times 7.6 \approx 1490$（FFT）+ $196 \times 384^2 \approx 2.89 \times 10^7$（MLP）

看起来 AFNO 更贵？注意这里 $N=196$ 太小，FFT 的优势不明显。当 $N=3136$（56x56 patch）时：
- 自注意力：$3136^2 \times 384 \approx 3.78 \times 10^9$
- AFNO：$3136 \times 12 \approx 37632$（FFT）+ $3136 \times 384^2 \approx 4.63 \times 10^8$

AFNO 比自注意力便宜约 8 倍。而且 $N$ 越大，优势越明显。

**(d) 直觉理解**

AFNO 的复杂度由两项组成：
- $N \log N$：FFT 的代价，随 $N$ 增长缓慢
- $ND^2$：频域处理的代价，随 $D$ 平方增长

这意味着 AFNO 适合 $N$ 大 $D$ 小的场景（高分辨率图像、小维度 token），而不适合 $N$ 小 $D$ 大的场景（少量高维 token）。

**(e) 边界情况**
- 当 $N=1$ 时，AFNO 退化为 MLP（FFT 是恒等映射）
- 当 $D=1$ 时，复杂度为 $O(N \log N + N) = O(N \log N)$
- 当 $N$ 极大时（如 $N=10^6$），FFT 的 $O(N \log N)$ 仍比 $O(N^2)$ 好得多

## 九、位置

### 前驱工作

1. **FNO [Li et al., 2021]**：傅里叶神经算子，在 PDE 求解中首次提出频域全局积分。AFNO 的核心思想直接继承自 FNO。
2. **ViT [Dosovitskiy et al., 2021]**：Vision Transformer，证明了 Transformer 在视觉任务上的有效性。AFNO 作为 ViT 的 token mixing 替代方案。
3. **MLP-Mixer [Tolstikhin et al., 2021]**：用 MLP 替代自注意力。AFNO 进一步用频域 MLP 替代空域 MLP。
4. **Weight Normalization [Salimans & Kingma, 2016]**：权重归一化技术。AFNO 将其扩展到复数域。

### 同期竞品

1. **GFNet [Rao et al., 2021]**：Global Filter Network，用 FFT + 逐元素乘积做 token mixing。AFNO 用 MLP 替代乘积，表达能力更强。
2. **Swin Transformer [Liu et al., 2021]**：窗口注意力 + shifted window。AFNO 用频域全局操作替代局部窗口操作。
3. **ConvMixer [Trockman & Kolter, 2022]**：用 depthwise 卷积做 token mixing。AFNO 用频域操作实现全局感受野。

### 后续影响

1. **SpectralGPT [2023]**：将频域操作扩展到视觉语言模型，用 AFNO 做跨模态 token mixing。
2. **FourierFormer [2023]**：将 AFNO 的频域 MLP 与自注意力结合，提出"傅里叶注意力"。
3. **Adaptive Fourier Convolution [2023]**：将 AFNO 的自适应频点处理思想引入卷积网络。
4. **Spectral Graph Transformer [2024]**：将 AFNO 的频域操作扩展到图数据，用图傅里叶变换做 token mixing。
5. **Wavelet Transformer [2023]**：用小波变换替代 FFT，解决 FFT 的固定分辨率问题。

## 十、局限

1. **高频信息丢失**：AFNO 只处理低频成分（默认保留前 50%），高频信息被丢弃。对于需要高频细节的任务（如超分辨率、纹理合成），这可能是一个问题。

2. **参数量随 $N$ 增长**：频域 MLP 的参数量是 $O(ND^2)$，当 $N$ 很大时（如高分辨率图像），参数量可能超过自注意力。虽然 FFT 节省了计算量，但参数量的增长可能带来过拟合。

3. **复数运算的硬件效率**：复数矩阵乘法在 GPU 上没有专门的优化，需要手动将实部和虚部分开处理，增加了实现复杂度和潜在的计算开销。

4. **固定网格限制**：AFNO 要求 token 排列在规则的 2D 网格上（$H \times W$），不能处理不规则排列的 token（如点云、图数据）。这限制了它在非视觉任务上的应用。

5. **与预训练模型的兼容性**：AFNO 替换自注意力后，无法直接加载预训练的 ViT 权重（因为参数结构完全不同）。这意味着需要从头训练，无法利用大规模预训练。

## 十一、小结

AFNO 提出了一个优雅的解决方案：用 FFT 将 token mixing 从空域搬到频域，在频域用逐频点 MLP 做全局交互，再变换回空域。核心优势是 $O(N \log N)$ 的复杂度和全局感受野，同时通过权重归一化和自适应频点处理保证了训练稳定性和表达能力。

实验结果表明，AFNO 在 ImageNet 分类和 ADE20K 语义分割上优于 Swin Transformer 和 GFNet 等同期工作。但它的主要贡献不是某个 SOTA 结果，而是提供了一种替代自注意力的新范式：频域 token mixing。

这个工作的更广泛影响是：它打开了"在频域做注意力"这个研究方向，后续的 SpectralGPT、FourierFormer 等工作都直接受其启发。虽然 AFNO 本身没有成为主流（Swin Transformer 和 ConvNeXt 仍然更流行），但它证明了频域操作在视觉 Transformer 中的可行性，为后来者铺平了道路。
