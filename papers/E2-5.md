# Adaptive Fourier Neural Operators: Efficient Token Mixers for Transformers (AFNO, 2022)

**论文：** [arXiv:2111.13587](https://arxiv.org/abs/2111.13587) · John Guibas, Morteza Mardani, Zongyi Li, Andrew Tao, Anima Anandkumar, Bryan Catanzaro · NVIDIA

**技术分支：** E2 — token mixing 机制 / E2-2 — 频域 mixing 范式

---

## 核心思路

AFNO 将 FNO（Fourier Neural Operator）适配为 Vision Transformer 的 token mixing 模块。相比 FNO，AFNO 的三个关键改进：

1. **Block-diagonal 权重**：在频域对每个频率的 channel mixing 矩阵施加 block-diagonal 结构，降低参数量
2. **自适应权重共享**：不再为每个频率独立学习权重，而是通过 soft-thresholding 实现频域稀疏化和权重共享
3. **谱域软阈值收缩**：去除信息量低的频率分量，提高效率

## 复杂度

- Self-attention: O(N²·d) 
- AFNO: O(N log N · d)，且显存是 O(N) 线性（不像 attention 的 O(N²)）
- 可处理 65K token 序列（Cityscapes 分割）

## 实验结果

- 在 Cityscapes 语义分割上使用 Segformer-B3 backbone，AFNO 可处理 65K 序列，优于其他高效 attention 机制
- Few-shot 分割上 AFNO 在效率和精度上都优于 self-attention
- 准线性复杂度，显存随序列长度线性增长
