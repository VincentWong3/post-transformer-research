## 一、引言：高分辨率视觉 Transformer 的困境

Vision Transformer（ViT）将图像切割为 patch 序列后，通过自注意力（self-attention）在 token 之间建立全局依赖关系。这种"token mixing"机制是 Transformer 能力的关键来源。然而，自注意力的复杂度为 O(N²d)，其中 N = h × w 为序列长度，d 为通道维数。当图像分辨率提高时（例如 Cityscapes 的 1024×2048），N 可达数万甚至数十万，自注意力的二次复杂度使其在计算和显存上不可承受。

为了解决这一瓶颈，研究者提出了多种近似方案：稀疏注意力、低秩近似、核方法、聚类等。但这些方法通常以降低精度换取效率。另一类工作则放弃图结构假设，转而利用傅里叶变换等几何工具进行 token mixing，如 GFNet 和 FNet。然而，GFNet 仅做深度可分离的全局卷积（depthwise global convolution），不做通道混合，表达能力受限；其参数量随输入分辨率增长，缺乏自适应能力。

本文提出 Adaptive Fourier Neural Operator（AFNO），将傅里叶神经算子（FNO）从偏微分方程求解领域引入视觉 Transformer，作为高效、可扩展的 token mixer。AFNO 的复杂度为准线性 O(Nd²/k + Nd log N)，显存为 O(N) 线性，在 Cityscapes 分割中可以处理 65K token 序列，比高效自注意力高 2% mIoU，在少样本分割中比自注意力快 30% 且精度持平或略优。

## 二、相关工作：三类高效 Token Mixer

### 2.1 基于图的 Mixer（Graph-Based）

以自注意力为核心，通过近似手段降低 O(N²) 复杂度：

- **稀疏注意力**：Sparse Transformer、Longformer 等强制稀疏注意力模式
- **低秩注意力**：Linformer、Long-Short Transformer（LS）通过低秩投影降低计算量
- **核方法**：Performer、Linear Transformer 用核函数近似注意力
- **聚类方法**：Reformer、Routing Transformer 将 token 分组后再计算

这些方法的主要缺点是精度与效率之间的权衡。

### 2.2 基于 MLP 的 Mixer（MLP-Based）

MLP-Mixer 完全放弃自注意力，用两个 MLP 分别在空间维和通道维做 mixing。ResMLP 和 gMLP 进一步改进。但 MLP 投影的复杂度仍为二次（O(N²)），且参数量随分辨率增长，不适合高分辨率场景。

### 2.3 基于傅里叶的 Mixer（Fourier-Based）

- **FNet**：用固定离散傅里叶变换（DFT）做 token mixing，无任何可学习参数，表达能力有限
- **Global Filter Networks（GFNet）**：学习可调的傅里叶滤波器，做 depthwise 全局卷积，但无通道混合，且滤波器参数量随输入分辨率增长
- **AFNO（本文）**：在傅里叶域做全通道混合的全局卷积，通过 block-diagonal 结构、自适应权重共享和稀疏化解决 FNO 在视觉任务中的局限性

### 2.4 算子学习（Operator Learning）

算子学习旨在学习函数空间之间的映射，广泛应用于 PDE 求解。DeepONet 和神经算子（Neural Operator）通过核积分实现函数映射。FNO 用 FFT 高效实现全局卷积，在 PDE 领域表现优异。本文的核心洞见是：**token mixing 本质上是连续函数空间中的全局卷积算子，可以借用 FNO 的框架来实现高效 mixing**。

## 三、从自注意力到全局卷积：数学形式化

### 3.1 自注意力作为核积分

输入张量 X ∈ ℝ^{h×w×d}，将序列索引记为 s, t ∈ [N]，自注意力可写为：

```
Att(X)[s] = Σ_{t=1}^{N} X[t] κ[s, t]
```

其中 κ[s, t] ∈ ℝ^{d×d} 是矩阵值核函数，由 softmax 后的注意力分数与值矩阵 W_v 共同定义。这一离散求和可推广为连续积分：

```
K(X)(s) = ∫_D κ(s, t) X(t) dt
```

这启发了算子学习视角：神经网络不再处理有限维向量，而是作用于函数空间上的算子。

### 3.2 全局卷积与格林核

当核函数只依赖于位移，即 κ(s, t) = κ(s - t) 时，核积分退化为全局卷积：

```
K(X)(s) = ∫_D κ(s - t) X(t) dt
```

这就是格林核（Green's kernel）假设——所有 token 之间的相互作用仅取决于它们的相对位置，而非绝对位置。这一假设天然地契合图像的平移等变性（translation equivariance），同时将复杂度从 O(N²) 降为近 O(N)。

### 3.3 卷积定理与傅里叶神经算子

根据卷积定理，全局卷积在傅里叶域等价于逐元素的乘法。对于连续情形：

```
K(X)(s) = F^{-1}[ F(κ) · F(X) ](s)
```

其中 · 为矩阵乘法。这正是傅里叶神经算子（Fourier Neural Operator, FNO）的核心公式。在离散网格上，用 DFT 替代连续傅里叶变换，得到离散 FNO。

## 四、以 FNO 作为 Token Mixer：可行性与局限性

### 4.1 离散 FNO 的三步操作

对 token 张量 X ∈ ℝ^{h×w×d}，FNO mixing 包含以下步骤：

1. **Token mixing（傅里叶变换）**：z_{m,n} = [DFT(X)]_{m,n}
2. **Channel mixing（频域加权）**：z̃_{m,n} = W_{m,n} z_{m,n}，其中 W ∈ ℂ^{h×w×d×d}
3. **Token demixing（逆变换）**：y_{m,n} = [IDFT(Z̃)]_{m,n}

### 4.2 残差项处理局部特征

DFT 假设图像是周期延拓的，对于真实图像（存在边缘、非周期边界），可以加入残差连接或局部卷积来补偿局部特征。

### 4.3 分辨率不变性

FNO 通过傅里叶基参数化 token 函数，因此不依赖于离散化分辨率。训练在某一分辨率完成后，可直接零样本超分辨率（zero-shot super-resolution）在更高分辨率上评估。

### 4.4 FNO 作为 Token Mixer 的固有缺陷

将 FNO 直接用于视觉任务面临三个问题：

1. **参数量过大**：每个频率模式 (m,n) 都有一个 d×d 的权重矩阵 W_{m,n}，总参数量 O(Nd²)，随分辨率线性增长
2. **静态权重**：W_{m,n} 训练后固定不变，不能像自注意力那样根据输入自适应调整
3. **缺乏稀疏归纳偏置**：图像在傅里叶域具有天然稀疏性（能量集中在低频），FNO 对所有频率一视同仁

AFNO 正是针对这三点逐一改进。

## 五、改进一：Block-Diagonal 结构——控制通道混合复杂度

### 5.1 动机

FNO 中每个 token 对应一个 d×d 的复数权重矩阵 W_{m,n}。当 d 很大时（ViT 中 d 可达 768 甚至更高），通道混合的计算量和参数量都难以承受。复杂度为 O(Nd²)。

### 5.2 方法

将 W_{m,n} 施加块对角结构（block-diagonal），分成 k 个大小均为 d/k × d/k 的块：

```
z̃_{m,n}^{(ℓ)} = W_{m,n}^{(ℓ)} z_{m,n}^{(ℓ)},  ℓ = 1, ..., k
```

每个块独立运算。这一结构有两个重要性质：

- **可并行化**：k 个块的乘法相互独立，可高效并行执行
- **可解释性**：每个块类似于 multi-head self-attention 中的一个 head，投影到数据的子空间中

### 5.3 特殊情况

当 k = d（块大小为 1）时，FNO 的通道混合部分（无 AFNO 的改进）退化为 GFNet 风格——每个通道独立做 global convolution，无通道混合。这表明 GFNet 是 AFNO 的一个特例，而 AFNO 通过块对角结构在通道混合与计算效率之间取得平衡。

复杂度从 O(Nd²) 降为 O(Nd²/k)，参数也从 O(Nd²) 降为 O(Nd²/k)。

---

## 六、改进二：自适应权重共享——让 Token 学会交互

### 6.1 动机

原始 FNO 的 W_{m,n} 是静态的，训练好后对所有输入样本固定不变，缺乏自适应能力。自注意力之所以强大，部分原因在于其权重是动态生成的（取决于 query-key 的点积）。

### 6.2 方法

用两层的共享 MLP 替代每个 token 独立的 W_{m,n}：

```
z̃_{m,n} = MLP(z_{m,n}) = W₂ σ(W₁ z_{m,n} + b₁) + b₂
```

关键点在于：**W₁, W₂, b₁, b₂ 在所有 token 之间共享**。这意味着：

- 参数量不再随 N 增长（从 O(Nd²) 降为 O(d² + d)）
- 不同 token 通过共享的 MLP 间接交互——MLP 学习到的映射函数统一应用于所有 token
- 保留了某种程度的自适应性：MLP 的输入 z_{m,n} 随输入图像变化，因此输出也动态调整

### 6.3 与自注意力的对比

自注意力的自适应来源于 query-key 的点积，复杂度 O(N²d)。AFNO 的自适应来源于共享 MLP 的输入依赖，复杂度 O(Nd²/k)。两者都实现了"权重随输入变化"，但 AFNO 付出了更低的代价。

消融实验（表 6）表明，将自适应权重替换为静态权重后，CelebA-Faces 少样本分割 mIoU 从 49.49 降至 44.04，降幅超过 5 个百分点，证实了自适应机制的关键作用。

## 七、改进三：软阈值稀疏化——利用傅里叶域的稀疏性

### 7.1 动机

自然图像在傅里叶域的能量分布高度集中，大部分能量集中在低频分量，高频分量通常幅度很小。如果对高频分量施加适当的稀疏约束，可以减少噪声、提升泛化。

### 7.2 方法

受 LASSO 回归启发，将通道混合建模为带 ℓ₁ 正则化的优化问题：

```
min_{z̃} ||z̃ - Wz||² + λ||z̃||₁
```

其闭式解为软阈值收缩操作（soft-thresholding and shrinkage）：

```
z̃_{m,n} = S_λ(W_{m,n} z_{m,n})
```

其中 S_λ(x) = sign(x) · max{|x| - λ, 0}，λ 为控制稀疏度的超参数。

### 7.3 效果

- **天然适应图像内容**：低频（能量大）不受影响，高频（能量小）被收缩到零
- **正则化作用**：稀疏化减少了模型的有效自由度，改善泛化
- **计算友好**：软阈值操作是逐元素的，没有额外开销

消融实验（图 4）显示，λ = 0.01 时在图像修复和少样本分割上取得最佳性能，λ 过大则信息丢失过多，λ = 0（无稀疏）则性能下降。

## 八、完整 AFNO Token Mixer 架构与复杂度分析

### 8.1 前向流程

AFNO token mixer 的完整前向流程如下（图 3 伪代码）：

```
def AFNO(x):
    bias = x
    x = RFFT2(x)                              # 实值→复值，取半平面
    x = x.reshape(b, h, w//2+1, k, d/k)       # 分块
    x = BlockMLP(x)                            # 共享 MLP（含批量矩阵乘法+ReLU）
    x = x.reshape(b, h, w//2+1, d)             # 合并通道
    x = SoftShrink(x, λ)                       # 软阈值收缩
    x = IRFFT2(x)                              # 逆变换
    return x + bias                            # 残差连接
```

其中 BlockMLP 是两层的复数域 MLP，权重在所有 token 间共享。

### 8.2 复杂度对比

| 模型 | 计算复杂度 | 参数复杂度 | 含义 |
|------|-----------|-----------|------|
| Self-Attention | O(N²d + 3Nd²) | O(3d²) | 图全局卷积 |
| GFN | O(Nd + Nd log N) | O(Nd) | Depthwise 全局卷积 |
| FNO | O(Nd² + Nd log N) | O(Nd²) | 全局卷积 |
| **AFNO（本文）** | **O(Nd²/k + Nd log N)** | **O((1+4/k)d² + 4d)** | **自适应全局卷积** |

关键指标：

- **准线性复杂度**：AFNO 的计算复杂度相对于序列长度 N 为准线性（Nd log N 主导），而非二次
- **线性显存**：AFNO 不需要存储 N×N 的注意力矩阵，显存增长为 O(N)
- **参数量不随分辨率增长**：共享 MLP 的参数量只与 d 和 k 有关，与 N 无关

## 九、实验一：ImageNet-1K 图像修复（Inpainting Pretraining）

### 9.1 实验设置

- **骨干网络**：ViT-B/4（patch size 4×4），12 层
- **任务**：随机掩码图像修复，MSE 损失
- **评价指标**：PSNR、SSIM
- **训练**：100 epoch，Adam 优化器

### 9.2 结果

| Mixer | Params | GFLOPs | Latency(sec) | SSIM | PSNR(dB) |
|-------|--------|--------|-------------|------|----------|
| Self-Attention | 87M | 357.2 | 1.2 | 0.931 | 27.06 |
| LS | 87M | 274.2 | 1.4 | 0.920 | 26.18 |
| GFN | 87M | 177.8 | 0.7 | 0.928 | 26.76 |
| **AFNO** | **87M** | **257.2** | **0.8** | **0.931** | **27.05** |

AFNO 在 PSNR（27.05）和 SSIM（0.931）上与自注意力几乎持平，但计算量（257.2 GFLOPs）远低于自注意力（357.2 GFLOPs），且推理延迟（0.8s）也低于自注意力（1.2s）。对比 LS 和 GFN，AFNO 在各项指标上全面领先。

这为下游少样本分割提供了良好的预训练基础。

## 十、实验二：少样本语义分割（Few-Shot Segmentation）

### 10.1 实验设置

- **任务**：1-shot 语义分割
- **数据集**：CelebA-Faces（人脸）、ADE-Cars（车辆）、LSUN-Cats（猫）
- **骨干**：ViT-B/4（使用图像修复预训练权重）
- **损失**：逐像素交叉熵
- **微调**：2000 epoch，学习率 10⁻³（AFNO/GFN/LS）/ 10⁻⁴（Self-Attention）

### 10.2 结果

| Mixer | Params | GFLOPs | LSUN-Cats | ADE-Cars | CelebA-Faces |
|-------|--------|--------|-----------|----------|-------------|
| Self-Attention | 87M | 357.2 | 35.57 | 49.26 | 56.91 |
| LS | 87M | 274.2 | 20.29 | 29.66 | 41.36 |
| GFN | 87M | 177.8 | 34.52 | 47.84 | 55.21 |
| **AFNO** | **87M** | **257.2** | **35.73** | **49.60** | **55.75** |

AFNO 在 3 个数据集的 2 个上（LSUN-Cats: 35.73, ADE-Cars: 49.60）超过了自注意力，在 CelebA-Faces 上仅低约 1 个百分点。与 GFN 相比，AFNO 在所有数据集上均大幅领先（+1～+2 mIoU）。

**效率优势**：AFNO 的计算量比自注意力少约 28%（257.2 vs 357.2 GFLOPs），推理速度更快（0.8s vs 1.2s），同时保持了可比甚至更优的精度。论文特别指出 AFNO 比自注意力**快约 30%**。

## 十一、实验三：Cityscapes 高分辨率语义分割

### 11.1 实验设置

- **任务**：高分辨率（1024×1024 裁剪）语义分割
- **骨干**：SegFormer-B3（4 阶段层次化结构，序列长度可达 65K）
- **数据集**：Cityscapes
- **对比方法**：Self-Attention（SA）、Efficient SA、LS、GFN、AFNO
- **注意**：标准 Self-Attention 因序列过长无法训练，故未纳入

### 11.2 结果

| Mixer | Params | Total GFLOPs | Mixer GFLOPs | mIoU |
|-------|--------|-------------|-------------|------|
| Efficient SA | 45M | 380.7 | 129.9 | 79.7 |
| LS | 45M | 409.1 | 85.0 | 80.5 |
| GFN | 45M | 363.4 | 2.6 | 80.4 |
| **AFNO-100%** | **45M** | **440.0** | **23.7** | **80.9** |
| **AFNO-25%** | **45M** | **429.0** | **12.4** | **80.4** |

AFNO-100%（保留所有频率模式）以 80.9 mIoU 击败了所有对比方法，比 Efficient SA 高 1.2 个百分点，比 LS 高 0.4 个点，比 GFN 高 0.5 个点。

AFNO-25% 仅保留 25% 的低频模式进行微调，mIoU 仍有 80.4，与 GFN 持平。这表明傅里叶域的高频截断对分割性能影响不大，验证了傅里叶域的稀疏性。

需要注意的是，总体 FLOPs 中 mixer 部分占比不大（23.7 GFLOPs），瓶颈在于 MLP 层。AFNO 的 mixer FLOPs（23.7）远低于 Efficient SA（129.9）和 LS（85.0），仅高于 GFN（2.6），但 GFN 不做通道混合。

### 11.3 多尺度对比

论文进一步在 Segformer B0-B3 各尺度上对比 AFNO、GFN 和 Efficient SA，并纳入 Swin 和 ResNet/MobileNetV2 基线。结果显示：

- AFNO 在所有 Segformer 骨干尺度上一致优于 GFN 和 Efficient SA
- AFNO + Segformer 一致优于 Swin Transformer 各尺度版本
- LS 和标准自注意力因半精度训练不稳定和二次显存问题未能参与多尺度对比

## 十三、消融实验：各组件的贡献量化

### 13.1 稀疏阈值 λ 的影响

改变 λ 从 0 到 10，在 ImageNet 图像修复（PSNR）和 CelebA-Faces 少样本分割（mIoU）上评估。

- λ = 0（无稀疏）：PSNR ≈ 25.4，mIoU ≈ 47
- λ = 0.01（最优）：PSNR ≈ 25.69，mIoU ≈ 49.49
- λ = 0.1：PSNR 和 mIoU 略降
- λ = 10（过度稀疏）：严重退化

这表明**适度的稀疏约束有助于提升性能**，傅里叶域的稀疏性是有效的归纳偏置。

### 13.2 块数 k 的影响

控制总参数量不变，改变块数 k（1 到 64）：

- k = 8 时在 ImageNet 修复 PSNR 上取得最佳
- k = 1 或 64 时性能下降

k = 1（全连接 W）过参数化，k = 64（接近 GFN）缺乏通道混合能力。AFNO 默认使用 k = 16 或 8。

### 13.3 自适应权重 vs 静态权重

| 变体 | Params | PSNR | CelebA-Faces mIoU |
|------|--------|------|-------------------|
| FNO | 16M | 24.80 | 39.27 |
| AFNO [Non-Adaptive] | 16M | 25.10 | 44.04 |
| AFNO [Hard Threshold 35%] | 16M | 23.58 | 34.17 |
| **AFNO（完整）** | **16M** | **25.69** | **49.49** |

- **AFNO vs FNO**：相同参数量下完整 AFNO 的 PSNR 高出 0.89 dB，mIoU 高出 10.2 个百分点
- **自适应 vs 静态**：自适应权重（完整 AFNO）比静态权重高 0.59 dB PSNR 和 5.45 mIoU
- **软阈值 vs 硬截断**：软阈值（完整 AFNO）比硬截断仅保留 35% 低频模式（Hard Threshold 35%）高出 2.11 dB PSNR 和 15.32 mIoU，说明软阈值的自适应稀疏化远优于简单的高频截断

## 十四、可视化分析：AFNO 学到了什么

### 14.1 Token 的谱聚类可视化

论文在 10 层 Transformer 的每一层对中间特征进行谱聚类（k-NN 核，k=10，聚类数=4），比较不同 mixer 的聚类效果：

- **自注意力（SA）**：聚类边界清晰，与图像中的物体边界高度一致
- **AFNO**：聚类质量与 SA 相当，在高层特征中能清晰区分不同物体区域
- **GFN**：聚类轮廓模糊，物体边界不清晰
- **LS（Long-Short Transformer）**：早期尚可，最后一层聚类已基本丢失物体结构

这一可视化结果与少样本分割中的性能排名一致：SA ≥ AFNO > GFN > LS。

### 14.2 稀疏掩码可视化

观察软阈值操作后的 token 幅度（log 域，λ = 0.1），并计算沿通道和块维度被置零的比例：

- 输入图像的傅里叶表示高度稀疏，大部分高频分量被收缩为零
- 掩码模式随网络层次变化：底层保留较多低频信息，高层可以更激进地压缩
- 这一发现说明**可以安全地截断大部分高频分量而不损失性能**（AFNO-25% 仅损失 0.05 mIoU 的实证也支持这一结论）

## 十五、总结与展望

### 15.1 核心贡献

1. **跨领域迁移**：将 PDE 领域的 FNO 引入视觉 Transformer，建立了算子学习与高效 token mixing 之间的理论桥梁
2. **三个针对性改进**：块对角结构（block-diagonal）控制通道混合复杂度、自适应权重共享增强表达能力、软阈值稀疏化利用傅里叶域的固有稀疏性
3. **极致的效率-精度平衡**：准线性复杂度 O(Nd²/k + Nd log N)、线性显存 O(N)，在 Cityscapes 上处理 65K token、比 Efficient SA 高 2% mIoU，少样本分割比自注意力快 30%

### 15.2 局限与未来方向

1. **DFT 的全局性限制**：DFT 假设全局周期延拓，不能很好地捕捉图像局部结构。论文提出可用小波变换（Wavelet transform）替代 DFT 以更好地兼顾局部与全局信息
2. **MLP 层成为瓶颈**：在 Cityscapes 实验中，AFNO 的 mixer FLOPs 远低于整个模型的 FLOPs，主要计算开销来自 MLP 层（而非 mixing 模块）。未来可探索更高效的 MLP 设计
3. **块数的选择**：虽然消融实验建议 k = 8 最优，但最佳块数可能随任务和骨干网络变化，需要自动化的超参数选择

### 15.3 影响与启示

AFNO 是"频域 mixing 范式"中的代表性工作之一（GFNet → FNO → AFNO 的发展脉络）。它展示了傅里叶域作为 token mixing 空间的巨大潜力：

- 通过卷积定理将 O(N²) 的空间域 mixing 降为 O(N log N) 的频域逐元素乘法
- 频域的稀疏性天然适配图像数据，可以大幅压缩冗余信息
- 分辨率不变性使得训练后可零样本迁移到更高分辨率

这一思路后来被 HPRN、SFNO、KAN 等后续工作继承和扩展，成为视觉 Transformer 高效化的重要分支之一。
