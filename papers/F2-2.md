# Neural Ordinary Differential Equations（2018）

**论文：** [arXiv](https://arxiv.org/abs/1806.07366) · Ricky T. Q. Chen, Yulia Rubanova, Jesse Bettencourt, David Duvenaud · NeurIPS 2018 Best Paper

## 一、先搞清楚"问题在哪里"

残差网络 ResNet 是 2015 年以来最成功的视觉架构之一。其核心公式是：

$$h_{t+1} = h_t + f(h_t, \theta_t)$$

每一层都在前一层的基础上加一个残差项 $f$。如果我们把 ResNet 的层数增加到 100、200、400 层，我们实际上是在**离散地**逼近一个连续变换。

一个反直觉的问题浮出水面：**如果残差网络本质上是在离散化一个连续系统，那我们为什么不能直接学习这个连续系统？**

## 二、旧路线的真正问题

标准残差网络（以及所有离散层网络）有三个根本性的限制：

1. **层数必须整数。** 你需要提前决定有多少层。你不能在"半层"停下来——如果你在中间停止前向传播，特征的质量会很差。而在连续系统中，你可以在任何时候停止（任何 t 值）。

2. **内存消耗随深度线性增长。** 训练时需要保存所有层的中间激活值用于反向传播。100 层 ResNet 需要保存 100 份计算图。如果网络更深，显存先于参数先爆。

3. **计算量与深度不可分离。** 推理时不管输入多简单，网络都要跑完全部层。连续 ODE 可以随着输入的难度自适应地增减求解步数。

## 三、这篇论文的核心思路

> 这篇论文的核心思路是：把残差网络视为常微分方程（ODE）的欧拉离散化，然后用 ODE solver 替代固定层数的网络，用 adjoint sensitivity 方法做反向传播。

```text
ResNet:     h_{t+1} = h_t + f(h_t)         ← 欧拉离散化
Neural ODE: dh/dt = f(h(t), t, θ)          ← 连续版本
            h_T = h_0 + ∫_0^T f(h(t), t) dt ← ODE solver 求解
```

## 五、核心模块逐个解释

### 1. ODE Solver 作为网络层

**为什么需要 ODE solver？** 残差网络用固定数量的欧拉步（每层一步），步长 = 1。连续系统需要更精确的积分——这就是 ODE solver 的职责。

**工作原理：** 给定初始状态 $h(0)$ 和参数化导数函数 $f(h,t,\theta)$，ODE solver 计算 $h(T) = h(0) + \int_0^T f(h(t),t,\theta) dt$。solver 内部可以自适应步长（如 Dormand-Prince 方法），在梯度变化快的区域用小步长，在平缓区域用大步长。

### 2. Adjoint Sensitivity Method 做反向传播

**为什么不能用标准反向传播？** 标准反向传播需要保存所有中间激活——但 ODE solver 的中间步数是自适应的、不可预测的。如果保存所有步的激活，内存和 ResNet 一样大。

**解决方案：** 使用 adjoint 方法。定义 adjoint 状态 $\mathbf{a}(t) = \partial L / \partial \mathbf{h}(t)$。通过求解另一个 ODE（从 T 到 0）来计算梯度，不需要保存任何中间状态：

$$\frac{d\mathbf{a}(t)}{dt} = -\mathbf{a}(t)^T \frac{\partial f(\mathbf{h}(t), t, \theta)}{\partial \mathbf{h}}$$

$$\frac{dL}{d\theta} = -\int_T^0 \mathbf{a}(t)^T \frac{\partial f(\mathbf{h}(t), t, \theta)}{\partial \theta} dt$$

**关键优势：** 这使得训练的内存复杂度从 $O(L)$ 降到 $O(1)$，与深度无关。你可以输入一个 1000 层等效的 ODE，训练时只消耗常数级显存。

## 六、核心公式与数学直觉

### 残差网络到 ODE 的连续化

离散残差连接：$h_{t+1} = h_t + f(h_t)$

如果 $t$ 是连续时间：$\frac{h(t+\Delta t) - h(t)}{\Delta t} = f(h(t))$，当 $\Delta t \to 0$：

$$\frac{dh(t)}{dt} = f(h(t), t, \theta)$$

### 前向传播

$$h(T) = h(0) + \int_0^T f(h(t), t, \theta) dt$$

**符号：**
- $h(0)$: 输入特征（或编码器输出）
- $h(T)$: ODE 终端状态→分类/预测
- $f$: 参数化的导数函数（一个神经网络）
- $T$: 积分时间（等效于"网络深度"）

### 反向传播（Adjoint Method）

$$a(t) = \frac{\partial L}{\partial h(t)}$$

$$\frac{da(t)}{dt} = -a(t)^T \frac{\partial f(h(t), t, \theta)}{\partial h}$$

**直觉：** adjoint 方程告诉你——当前隐藏状态的梯度如何随时间反向演化。这等价于反向传播，但不需要存中间状态。

## 十、局限

1. **推理速度慢。** ODE solver 的数值积分耗时远大于等价的离散残差网络。
2. **表达能力受限。** 标准 Neural ODE 不能表达恒等映射（$dh/dt=0$ 的解太特殊）。后续 Augmented Neural ODE 通过在输入空间扩充维度解决了这个问题。
3. **训练不稳定。** ODE solver 的数值误差会累积，需要调节 tolerance 参数。
4. **大规模应用验证不足。** 在 ImageNet 上还没有替代 ResNet。

## 十一、小结

Neural ODE 的核心贡献是：证明了残差网络可以自然地被解释为 ODE 离散化，从而引入连续深度建模这一新范式。它的 adjoint 方法将训练内存降至 $O(1)$，理论上可以训练任意"深度"的网络。
