**论文：** [arXiv](https://arxiv.org/abs/2010.08895) · Zongyi Li, Nikola Kovachki, Kamyar Azizzadenesheli, Burigede Liu, Kaushik Bhattacharya, Andrew Stuart, Anima Anandkumar · 2020

## 一、先搞清楚坑在哪

偏微分方程（PDE）的数值求解是科学计算的核心问题。传统方法如有限元（FEM）和有限差分（FDM）需要将空间离散化成网格，然后在每个时间步求解大规模线性系统。对于参数化PDE——即同一类PDE但参数（如流体粘度、材料属性）变化的情况——传统方法每次改变参数都需要重新求解，计算成本极高。

举个具体例子：模拟不同雷诺数下的圆柱绕流，传统求解器需要为每个雷诺数重新划分网格、组装刚度矩阵、迭代求解。这通常需要数小时甚至数天。

深度学习社区尝试用神经网络直接学习PDE的解算子（operator），即从参数空间到解空间的映射。但已有的方法存在根本性缺陷。

## 二、已有方法的真正问题

### 2.1 基于网格的方法

DeepONet [Lu et al., 2019] 和类似方法将PDE参数在固定网格点上的值作为输入。问题在于：

1. **网格依赖性**：训练时固定的网格分辨率，无法泛化到不同分辨率的输入
2. **局部性偏差**：卷积操作天然是局部的，而PDE的解往往依赖全局信息
3. **离散化不一致**：物理规律是连续的，但网络在离散网格上学习，导致离散化误差

### 2.2 图神经网络方法

GNN-based方法 [Pfaff et al., 2020] 将网格视为图，在节点间传递消息。但：

1. **计算开销大**：消息传递的复杂度随边数增长
2. **图构建困难**：不规则网格的图构建本身是个难题
3. **分辨率泛化差**：改变网格拓扑需要重新训练

### 2.3 核心痛点

所有现有方法都受限于**离散化**。它们学习的是"在特定网格上的映射"，而不是"连续函数空间之间的算子"。这意味着：

- 训练和测试必须使用相同分辨率的网格
- 无法利用多尺度信息
- 计算复杂度随分辨率增长过快

## 三、FNO的核心思路

Fourier Neural Operator（FNO）的核心洞察是：**在傅里叶空间进行全局卷积**。

一句话概括：FNO将PDE的解算子参数化为一组傅里叶变换域上的线性变换，通过快速傅里叶变换（FFT）实现高效计算，从而学习连续函数空间之间的映射。

**高层面板**：

```
输入函数 a(x) ──→ 提升维度 ──→ 傅里叶层 × N ──→ 投影到输出 ──→ 解函数 u(x)
                     │
                     └── 每个傅里叶层：
                         1. 傅里叶变换（FFT）
                         2. 线性变换（截断高频）
                         3. 逆傅里叶变换（IFFT）
                         4. 残差连接 + 激活函数
```

## 四、网络架构详解

### 4.1 整体架构

如图1所示，FNO的整体流程如下：

```
输入 a(x) ∈ ℝ^{d_a}
    │
    ▼
P: 局部提升层（全连接）
    │
    ▼
    ┌─────────────────────────────────────────────┐
    │  v_0(x) ∈ ℝ^{d_v}                          │
    │                                             │
    │  for t = 1 to T:                            │
    │      v_t(x) = σ( W·v_{t-1}(x) +             │
    │                𝒦·v_{t-1}(x) )               │
    │      ┌───────────────┐                      │
    │      │ 𝒦  = IFFT(    │                      │
    │      │       R·FFT(  │                      │
    │      │        v(x)   │                      │
    │      │       )       │                      │
    │      │     )         │                      │
    │      └───────────────┘                      │
    │                                             │
    └─────────────────────────────────────────────┘
    │
    ▼
Q: 局部投影层（全连接）
    │
    ▼
输出 u(x) ∈ ℝ^{d_u}
```

**图1：FNO整体架构。输入函数a(x)通过提升层P映射到高维空间，经过T个傅里叶层（每个包含傅里叶变换、线性变换R、逆傅里叶变换和残差连接），最后通过投影层Q得到输出u(x)。**

### 4.2 前向过程详解

**Step 1: 输入编码**

输入是一个函数 $a(x)$，在离散网格上采样得到值。例如在1D问题中，输入形状为 $(batch\_size, grid\_size, d_a)$，其中 $d_a$ 是输入通道数。

**Step 2: 提升层 P**

通过一个浅层全连接网络（通常是2-3层MLP）将输入映射到高维特征空间：
$$v_0(x) = P(a(x))$$
- 输入形状：$(batch\_size, grid\_size, d_a)$
- 输出形状：$(batch\_size, grid\_size, d_v)$，其中 $d_v$ 是隐层维度（论文中常用64-128）

**Step 3: 傅里叶层**

每个傅里叶层的核心操作是：
$$v_{t+1}(x) = \sigma\left(W v_t(x) + \mathcal{K}(v_t)(x)\right)$$

其中：
- $W$：局部线性变换（$1 \times 1$ 卷积）
- $\mathcal{K}$：全局卷积算子（傅里叶域实现）
- $\sigma$：激活函数（ReLU/GELU）

**傅里叶算子 $\mathcal{K}$ 的详细计算**：

```
输入 v(x) ∈ ℝ^{d_v}，在 n 个网格点上采样
    │
    ▼
FFT: v̂(k) = ℱ(v(x)) ∈ ℂ^{n × d_v}
    │  （对每个通道独立做FFT）
    ▼
截断：保留前 k_max 个傅里叶模式
    │  v̂_trunc ∈ ℂ^{k_max × d_v}
    ▼
线性变换：R ∈ ℂ^{d_v × d_v × k_max}
    │  (v̂_out)_i = Σ_j R_{ij} · (v̂_trunc)_j
    │  输出形状：ℂ^{k_max × d_v}
    ▼
填充零回到 n 个模式
    │  v̂_padded ∈ ℂ^{n × d_v}
    ▼
IFFT: 𝒦(v)(x) = ℱ^{-1}(v̂_padded) ∈ ℝ^{n × d_v}
```

**图2：傅里叶算子𝒦的计算流程。输入函数通过FFT变换到频域，截断高频模式后乘以可学习的复数权重矩阵R，再通过IFFT变换回空间域。**

**为什么截断高频？** 
- 计算效率：保留所有n个模式的复杂度是 $O(n^2)$，截断到 $k_{max}$ 后复杂度为 $O(n \log n + k_{max} d_v^2)$
- 正则化效果：高频通常对应噪声，截断起到隐式正则化作用
- 物理意义：PDE的解通常由低频模式主导

**Step 4: 投影层 Q**

经过T个傅里叶层后，通过另一个浅层全连接网络将特征映射到输出：
$$u(x) = Q(v_T(x))$$
- 输入形状：$(batch\_size, grid\_size, d_v)$
- 输出形状：$(batch\_size, grid\_size, d_u)$

### 4.3 为什么在傅里叶域？

这是FNO最核心的设计选择，理由有三：

1. **全局感受野**：傅里叶变换的基函数（正弦波）是全局的，一次变换就能捕获整个空间域的信息。相比之下，CNN需要堆叠很多层才能获得全局信息。

2. **平移等变性**：傅里叶变换天然具有平移等变性——输入平移对应频域中的相位旋转。这对于PDE问题非常关键，因为物理规律是平移不变的。

3. **分辨率不变性**：在傅里叶域中，我们可以处理任意分辨率的输入。训练时用64×64网格，测试时可以用256×256网格，因为FFT可以处理任意长度的序列。

## 五、核心创新点

### 5.1 傅里叶域参数化

**(a) 痛点与动机**

传统神经网络（如CNN）在空间域做卷积，每个卷积核只看到局部邻域。对于PDE问题，解在空间上存在长程依赖（如流体中的涡旋传播），局部卷积需要很多层才能传播信息。

DeepONet使用点乘运算，虽然可以处理全局信息，但计算复杂度是 $O(n^2)$。

**(b) 方案细节**

将卷积算子参数化为傅里叶域中的乘法：
$$\mathcal{K}(v)(x) = \mathcal{F}^{-1}\left(R \cdot \mathcal{F}(v)\right)(x)$$

其中 $R$ 是一个可学习的复数权重矩阵，$\mathcal{F}$ 是傅里叶变换。

**(c) 为什么有效**

- 傅里叶域中的乘法对应空间域中的全局卷积
- FFT的复杂度是 $O(n \log n)$，比 $O(n^2)$ 的全局注意力机制快得多
- 截断高频（保留 $k_{max}$ 个模式）进一步降低复杂度

**(d) 与Related Work的关系**

- **Gaussian Processes** [Rasmussen, 2006]：也使用傅里叶特征，但FNO是可学习的深度架构
- **Graph Neural Operators** [Li et al., 2020]：在图上做谱卷积，但FNO在规则网格上使用FFT更高效
- **DeepONet** [Lu et al., 2019]：使用点乘运算，复杂度 $O(n^2)$，FNO通过FFT降到 $O(n \log n)$

**(e) 如果去掉会怎样**

从表3（论文）可以看到，去掉傅里叶层（只用局部MLP）的FNO在达西流问题上相对L2误差从0.010上升到0.032，性能下降3倍。

### 5.2 分辨率不变性

**(a) 痛点与动机**

传统方法训练在固定分辨率上，测试时改变分辨率需要重新训练或插值。插值会引入额外误差。

**(b) 方案细节**

FNO的傅里叶层可以处理任意长度的输入序列。在频域中，不同分辨率对应不同的采样点数，但可学习的参数 $R$ 只依赖于截断模式数 $k_{max}$，与输入分辨率无关。

训练时：网格 $64 \times 64$，$k_{max} = 12$
测试时：网格 $256 \times 256$，$k_{max} = 12$（只学习前12个模式）

**(c) 为什么有效**

- 物理规律是连续的，FNO学习的是连续算子，不是离散映射
- 傅里叶模式是全局的，低分辨率输入中的低频模式在高分辨率输入中同样存在
- 截断操作确保模型不学习特定于分辨率的伪影

**(d) 与Related Work的关系**

- **PCANN** [Bhattacharya et al., 2020]：使用PCA降维，也具有一定分辨率不变性，但线性变换限制了表达能力
- **U-Net** [Ronneberger et al., 2015]：通过池化和上采样处理多尺度，但需要固定输入尺寸

**(e) 如果去掉会怎样**

论文表2显示，在纳维-斯托克斯方程问题上，训练在64×64网格上的FNO可以直接在256×256网格上测试，相对误差仅从0.0086上升到0.0091。如果去掉傅里叶层，模型完全无法泛化到不同分辨率。

### 5.3 端到端学习

**(a) 痛点与动机**

传统方法（如有限元）需要手动设计基函数和求解策略。DeepONet需要先验地选择传感器位置。

**(b) 方案细节**

FNO直接从输入函数（在任意网格上采样）学习到输出函数的映射，整个过程是端到端可微的。

**(c) 为什么有效**

- 不需要手动设计特征或基函数
- 梯度可以反向传播到所有参数，包括傅里叶权重 $R$ 和局部变换 $W$
- 可以处理不规则网格（通过插值到规则网格）

**(d) 与Related Work的关系**

- **Physics-Informed Neural Networks (PINNs)** [Raissi et al., 2019]：需要PDE的表达式作为损失函数
- **FNO**：完全数据驱动，不需要PDE表达式

**(e) 如果去掉会怎样**

如果不端到端学习，而是先提取特征再学习映射（如PCANN），性能会显著下降。论文表1显示，在伯格斯方程上，FNO的误差（0.0017）比PCANN（0.0037）低一个数量级。

## 六、公式详解

### 6.1 核心算子公式

$$\mathcal{K}(v)(x) = \mathcal{F}^{-1}\left(R \cdot \mathcal{F}(v)\right)(x)$$

**(a) 符号定义**

- $v(x)$：输入函数，定义在空间域 $x \in D \subset \mathbb{R}^d$
- $\mathcal{F}$：傅里叶变换，将函数从空间域变换到频域
- $\mathcal{F}^{-1}$：逆傅里叶变换
- $R$：可学习的复数权重张量，形状为 $(d_v \times d_v \times k_{max})$，其中 $d_v$ 是特征通道数，$k_{max}$ 是保留的傅里叶模式数
- $\cdot$：在频域中的矩阵乘法（对每个傅里叶模式做）

**(b) 公式来源**

这个公式来自卷积定理：空间域中的卷积等价于频域中的逐点乘法。标准形式为：
$$(f * g)(x) = \mathcal{F}^{-1}(\hat{f} \cdot \hat{g})$$

FNO将卷积核 $g$ 参数化为频域中的可学习矩阵 $R$。

**(c) 推导过程**

标准卷积：$(f * g)(x) = \int_D f(x-y)g(y)dy$

离散化后，对 $n$ 个网格点，复杂度 $O(n^2)$。

利用卷积定理：
$$\mathcal{F}(f * g) = \mathcal{F}(f) \cdot \mathcal{F}(g)$$

令 $\hat{g} = R$（可学习），则：
$$\mathcal{F}(\mathcal{K}(v)) = R \cdot \mathcal{F}(v)$$

两边逆变换得到：
$$\mathcal{K}(v) = \mathcal{F}^{-1}(R \cdot \mathcal{F}(v))$$

**(d) 直觉理解**

直观上，这个公式在做三件事：
1. **分解**：FFT将函数分解成不同频率的正弦波
2. **调制**：$R$ 决定每个频率成分如何被放大/缩小/相位偏移
3. **重构**：IFFT将调制后的频率成分重新组合成空间域函数

**具体数值例子**：

假设在1D问题上，网格点 $n=8$，特征通道 $d_v=2$，保留 $k_{max}=4$ 个模式。

输入 $v$ 形状：$(8, 2)$

Step 1: FFT得到 $\hat{v}$，形状 $(8, 2)$（复数）
Step 2: 截断，保留前4个模式，形状 $(4, 2)$
Step 3: 对每个模式 $k=0,...,3$：
$$\hat{v}_{out}[k] = R[k] \cdot \hat{v}[k]$$
其中 $R[k]$ 是 $2 \times 2$ 的复数矩阵

例如 $k=0$（直流分量）：
$$R[0] = \begin{bmatrix} 1+0i & 0.5+0i \\ 0+0i & 1+0i \end{bmatrix}$$
$$\hat{v}[0] = \begin{bmatrix} 3+0i \\ 2+0i \end{bmatrix}$$
$$\hat{v}_{out}[0] = \begin{bmatrix} 1\cdot(3+0i) + 0.5\cdot(2+0i) \\ 0\cdot(3+0i) + 1\cdot(2+0i) \end{bmatrix} = \begin{bmatrix} 4+0i \\ 2+0i \end{bmatrix}$$

Step 4: 填充零回到8个模式
Step 5: IFFT得到输出

**(e) 边界情况**

- $k_{max} = 0$：只保留直流分量，模型退化为全局平均池化
- $k_{max} = n$：保留所有模式，复杂度 $O(n^2)$，失去FFT的效率优势
- 输入分辨率 $n < k_{max}$：截断到 $n$，所有模式都被保留
- 输入维度 $d=1$：FFT退化为1D FFT
- 输入维度 $d=2$：使用2D FFT，$R$ 形状变为 $(d_v \times d_v \times k_{max}^2)$

### 6.2 傅里叶层更新公式

$$v_{t+1}(x) = \sigma\left(W v_t(x) + \mathcal{F}^{-1}\left(R \cdot \mathcal{F}(v_t)\right)(x)\right)$$

**(a) 符号定义**

- $v_t(x)$：第 $t$ 层的特征图
- $W$：局部线性变换，形状 $d_v \times d_v$（实现为 $1 \times 1$ 卷积）
- $\sigma$：激活函数（论文中使用GELU）

**(b) 公式来源**

这是神经算子框架 [Li et al., 2020] 的实例化。标准形式为：
$$v_{t+1} = \sigma(W v_t + \mathcal{K}(v_t))$$

其中 $\mathcal{K}$ 是核积分算子。

**(c) 推导过程**

公式包含两项：

1. **局部项** $W v_t(x)$：每个空间位置的独立线性变换
   - 作用：捕获局部特征，类似 $1 \times 1$ 卷积
   - 复杂度：$O(n d_v^2)$

2. **全局项** $\mathcal{K}(v_t)(x)$：通过傅里叶域实现的全局卷积
   - 作用：捕获长程依赖
   - 复杂度：$O(n \log n + k_{max} d_v^2)$

两项相加后通过非线性激活。

**(d) 直觉理解**

这个公式可以理解为"局部+全局"的特征融合：

- **局部项**：$W$ 像是一个"放大镜"，关注每个点本身的特征
- **全局项**：$\mathcal{K}$ 像是一个"望远镜"，看到整个空间的结构

两者结合，模型既能捕捉精细的局部细节，又能理解全局的物理模式。

**具体数值例子**：

假设 $d_v=2$，$n=64$，$k_{max}=12$

局部项 $W$：$2 \times 2$ 矩阵，复杂度 $O(2^2 \cdot 64) = 256$ 次乘法
全局项 $\mathcal{K}$：FFT $O(64 \log 64) \approx 384$ + 矩阵乘 $O(12 \cdot 2^2) = 48$ ≈ 432 次操作

总复杂度约 688 次操作，远小于 $O(n^2) = 4096$ 的全局注意力。

**(e) 边界情况**

- $W = 0$：只有全局信息，失去局部细节
- $R = 0$：只有局部信息，失去长程依赖
- $\sigma$ = 恒等映射：模型退化为线性算子
- $T=0$（没有傅里叶层）：模型退化为浅层MLP

### 6.3 损失函数

$$\mathcal{L}(u, u^*) = \frac{1}{N} \sum_{i=1}^N \frac{\|u_i - u_i^*\|_2}{\|u_i^*\|_2}$$

**(a) 符号定义**

- $u_i$：模型预测的第 $i$ 个样本的解
- $u_i^*$：第 $i$ 个样本的真实解（来自数值求解器）
- $\|\cdot\|_2$：L2范数
- $N$：训练样本数

**(b) 公式来源**

这是相对L2误差，也称为归一化均方根误差（NRMSE）。在PDE学习中广泛使用，因为不同问题的解的量级可能差异很大。

**(c) 推导过程**

标准MSE损失：$\frac{1}{N} \sum \|u_i - u_i^*\|_2^2$

问题：如果 $u^*$ 的量级是 $10^6$，误差 $10^3$ 看起来很大；如果 $u^*$ 的量级是 $10^{-3}$，误差 $10^{-3}$ 看起来很小。

相对L2损失除以真实解的范数，使误差归一化：
$$\text{相对误差} = \frac{\|u - u^*\|_2}{\|u^*\|_2}$$

这个值在0（完美预测）到无穷大之间，通常小于1表示预测优于零预测。

**(d) 直觉理解**

相对L2损失衡量的是"预测相对于真实解的百分比误差"：

- 0.01 表示平均相对误差 1%
- 0.1 表示平均相对误差 10%
- >1 表示预测比零向量还差

**具体数值例子**：

假设真实解 $u^* = [1, 2, 3, 4]$，预测 $u = [1.1, 1.9, 3.2, 3.8]$

$$\|u - u^*\|_2 = \sqrt{(0.1)^2 + (-0.1)^2 + (0.2)^2 + (-0.2)^2} = \sqrt{0.1} \approx 0.316$$
$$\|u^*\|_2 = \sqrt{1^2 + 2^2 + 3^2 + 4^2} = \sqrt{30} \approx 5.477$$
$$\text{相对误差} = \frac{0.316}{5.477} \approx 0.0577 = 5.77\%$$

- 所有样本的 $u^*$ 量级相同：相对L2等价于标准MSE的缩放版本

### 8.1 核心实现（来自GitHub仓库）

```python
# FNO层实现（简化版）
class FourierLayer(nn.Module):
    def __init__(self, modes, width):
        super().__init__()
        self.modes = modes  # k_max
        self.width = width  # d_v
        
        # 可学习的傅里叶权重（复数）
        self.weights = nn.Parameter(
            torch.randn(width, width, modes, 2)  # 2 for real/imag
        )
        # 局部线性变换
        self.linear = nn.Linear(width, width)
        
    def forward(self, x):
        # x shape: (batch, n, width) for 1D
        # FFT
        x_ft = torch.fft.rfft(x)
        
        # 截断高频
        x_ft_trunc = x_ft[:, :self.modes, :]
        
        # 复数矩阵乘法
        weight = torch.view_as_complex(self.weights)
        out_ft = torch.einsum('bim,ijm->bjm', x_ft_trunc, weight)
        
        # 填充并IFFT
        out_ft_padded = torch.zeros_like(x_ft)
        out_ft_padded[:, :self.modes, :] = out_ft
        out = torch.fft.irfft(out_ft_padded)
        
        # 残差连接 + 激活
        return F.gelu(out + self.linear(x))
```

### 8.2 代码中的设计细节

**1. 使用rfft而非fft**：对于实值输入，rfft只计算正频率，将计算量减半。

**2. 复数权重的处理**：代码中复数权重存储为形状 (width, width, modes, 2) 的张量，最后一维是实部和虚部。训练时使用 `torch.view_as_complex` 转换为复数。

**3. 截断策略**：只保留前 modes 个低频模式。在1D中，modes=12；在2D中，modes=12×12=144。

**4. 提升/投影网络**：使用简单的2层MLP，中间维度是隐层维度的2倍。

**5. 归一化**：在傅里叶层之间没有BatchNorm或LayerNorm，这与CNN不同。

**6. 训练技巧**：使用Adam优化器，学习率0.001，权重衰减1e-4。没有使用学习率调度。

## 十、局限

### 10.1 边界处理

**问题**：FFT假设周期边界条件。对于非周期边界（如Dirichlet边界），在边界附近会出现Gibbs现象。

**影响**：在处理复杂几何边界时精度下降。

**论文承认**：是，论文在实验部分讨论了这个问题。

**后续工作**：GeoFNO [Li et al., 2022] 通过变形映射解决不规则边界问题。

### 10.2 高频信息丢失

**问题**：截断高频模式意味着丢失细节信息。对于具有sharp interface或激波的PDE问题，高频信息很重要。

**影响**：在解不光滑的问题上精度受限。

**论文承认**：是，论文提到这是设计权衡。

**后续工作**：小波神经算子 [Gupta et al., 2021] 使用多分辨率分析保留更多细节。

### 10.3 网格规则性要求

**问题**：FFT要求规则网格。对于不规则网格（如自适应网格），需要先插值到规则网格。

**影响**：增加预处理开销，且插值引入误差。

**论文承认**：是，论文提到这是未来工作方向。

**后续工作**：MeshFreeFNO [Li et al., 2022] 直接处理不规则网格。

### 10.4 大尺度3D问题

**问题**：3D FFT的复杂度是 $O(n \log n)$，但 $n = N_x \times N_y \times N_z$ 增长很快。

**影响**：对于高分辨率3D问题，内存和计算开销仍然很大。

**论文承认**：部分承认，论文在附录讨论了复杂度。

**后续工作**：Factorized FNO [Tran et al., 2022] 将3D FFT分解为三个1D FFT。

### 10.5 长期预测稳定性

**问题**：自回归预测（用预测值作为下一步输入）时，误差会累积。

**影响**：长期预测精度下降，可能发散。

**论文承认**：是，论文在纳维-斯托克斯实验中讨论了这个问题。

**后续工作**：Latent Evolution FNO [Li et al., 2022] 在潜空间做时间步进。

## 十一、小结

Fourier Neural Operator 是深度学习求解偏微分方程领域的一个里程碑工作。它抓住了PDE问题的核心——物理规律是连续的，因此求解算子也应该是连续的。通过在傅里叶域参数化全局卷积算子，FNO同时解决了三个关键问题：

1. **计算效率**：FFT将全局卷积的复杂度从 $O(n^2)$ 降到 $O(n \log n)$
2. **分辨率泛化**：学习连续算子，天然支持不同分辨率的输入和输出
3. **物理一致性**：傅里叶变换的平移等变性符合物理规律

FNO在多个经典PDE问题上取得了SOTA结果，比传统数值方法快数万倍，比深度学习方法精度高一个数量级。

**更广泛的影响**：FNO开启了"神经算子"这个研究方向，将科学计算与深度学习更紧密地结合起来。后续工作将FNO扩展到天气预报（FourCastNet）、地震模拟、材料设计等实际应用，证明了这一范式的巨大潜力。FNO也被认为是"AI for Science"领域的代表性工作之一。
