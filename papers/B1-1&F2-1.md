# Deep Residual Learning for Image Recognition（2015）

**论文：** [arXiv](https://arxiv.org/abs/1512.03385) · Kaiming He, Xiangyu Zhang, Shaoqing Ren, Jian Sun · Microsoft Research · CVPR 2016 **Best Paper**

---

## 一、先搞清楚坑在哪

2015 年之前，卷积神经网络已经证明了"越深越好"——从 AlexNet（8 层，2012）到 VGG（19 层，2014）再到 GoogleNet（22 层，2014），网络越深，ImageNet 精度越高。但人们发现一个奇怪的现象：**当网络深度继续增加（超过 20-30 层），训练误差反而上升了**。

这不是过拟合——训练误差上升，测试误差也上升，且网络在训练集上的 loss 比浅层网络高。这意味着深层网络根本**优化不动**。

## 二、深层网络的真正问题——退化

作者做了关键实验：在一个 20 层网络后面堆叠更多层（变成 56 层），这些新增的层在理想情况下至少可以学到恒等映射（identity mapping，即输出等于输入），这样深层网络的性能至少不差于浅层网络。

但实际观察是：**56 层的训练/测试误差都比 20 层高**。这说明优化器无法让网络学到恒等映射——深层网络不是"表达能力不够"，而是"优化太难"。

这个问题和梯度消失/爆炸不同。梯度消失/爆炸已经被 BN（Batch Normalization）和合适的初始化部分解决了。退化是更深层的问题：**随着层数增加，网络的优化曲面越来越崎岖，SGD 难以找到好的解**。

## 三、ResNet 的核心思路

一句话：**在传统堆叠的层上添加一个"捷径连接"（shortcut connection），让网络学习残差映射 $F(x) = H(x) - x$ 而非直接映射 $H(x)$。**

如果目标映射是恒等映射（$H(x) = x$），让网络去学 $F(x) = 0$ 比学 $H(x) = x$ 容易得多——因为权重初始化接近 0，$F(x)$ 天然接近 0，所以 $F(x) + x$ 天然接近恒等映射。

## 四、网络架构详解

### 残差块（Residual Block）

```
         x
         │
    ┌────┴────┐
    │ Conv 3×3│
    │ BN      │
    │ ReLU    │
    │ Conv 3×3│
    │ BN      │
    └────┬────┘
         │
    ┌────┴────┐
    │  F(x)   │
    └────┬────┘
         │
x ───────┼── shortcut (identity mapping)
         │  +
         ▼
       ReLU
         │
         x'
```

数学表示：

$$y = F(x, \{W_i\}) + x$$
$$x' = \text{ReLU}(y)$$

其中 $F(x, \{W_i\})$ 是残差映射（两个 $3\times3$ 卷积 + BN + ReLU），$x$ 是 shortcut 连接的输入。

### 维度匹配问题

当残差块的输入和输出维度不一致时（channel 数不同或空间分辨率不同），shortcut 不能直接相加。两种处理方式：

1. **Projection shortcut**：用 $1\times1$ 卷积 $\text{Ws}$ 调整维度：$y = F(x, \{W_i\}) + W_s x$
2. **Zero-padding shortcut**：对短出的维度补零（论文实验证明 projection 略好，但零填充更省参数）

### 整体架构（ResNet-34 vs ResNet-50）

```
           ResNet-34                       ResNet-50
   Input 224×224×3                   Input 224×224×3
         │                                 │
   ┌─────▼──────┐                  ┌───────▼───────┐
   │ Conv7×7,64 │                  │ Conv7×7,64    │
   │ stride=2   │                  │ stride=2      │
   │ BN, ReLU   │                  │ BN, ReLU      │
   └─────┬──────┘                  │ MaxPool 3×3   │
         │                         └───────┬───────┘
   ┌─────▼────────────────────┐   ┌─────────▼───────────┐
   │ Conv2_x:  [3×3,64] ×3   │   │ Conv2_x: [1×1,64]   │
   │           [3×3,64]       │   │          [3×3,64] ×3│
   └─────┬────────────────────┘   │          [1×1,256]  │
         │                        └─────────┬───────────┘
   ┌─────▼────────────────────┐   ┌─────────▼───────────┐
   │ Conv3_x: [3×3,128] ×4    │   │ Conv3_x:[1×1,128]   │
   │          [3×3,128]       │   │         [3×3,128] ×4│

         │                        └─────────┬───────────┘
   ┌─────▼────────────────────┐   ┌─────────▼───────────┐
   │ Conv4_x: [3×3,256] ×6    │   │ Conv4_x:[1×1,256]   │
   │          [3×3,256]       │   │         [3×3,256] ×6│
   └─────┬────────────────────┘   │         [1×1,1024]  │
         │                        └─────────┬───────────┘
   ┌─────▼────────────────────┐   ┌─────────▼───────────┐
   │ Conv5_x: [3×3,512] ×3    │   │ Conv5_x:[1×1,512]   │
   │          [3×3,512]       │   │         [3×3,512] ×3│
   └─────┬────────────────────┘   │         [1×1,2048]  │
         │                        └─────────┬───────────┘
         ▼                                 ▼
   Average Pool (7×7)                Average Pool (7×7)
   1000-d fc, softmax                1000-d fc, softmax
```

**Bottleneck 设计（ResNet-50/101/152）**：用 `1×1 → 3×3 → 1×1` 替代 `3×3 → 3×3`。第一层 1×1 降维，3×3 在低维空间做变换，最后一层 1×1 恢复维度。总参数量减少，且可以堆叠更深。

### 前向过程（以 ResNet-50 为例）

**Stage 1 — Stem：** Conv7×7 stride=2（224→112），BN，MaxPool 3×3 stride=2（112→56）。输出 56×56×64。

**Stage 2 — Conv2_x：** 3 个 bottleneck，每个 `1×1(64)→3×3(64)→1×1(256)`。空间分辨率 56×56，channel 256。第一个 bottleneck 因为输入是 64 channel，输出 256，需要 projection shortcut。

**Stage 3 — Conv3_x：** 4 个 bottleneck，`1×1(128)→3×3(128)→1×1(512)`。第一个 bottleneck stride=2（56→28）。输出 28×28×512。

**Stage 4 — Conv4_x：** 6 个 bottleneck，`1×1(256)→3×3(256)→1×1(1024)`。第一个 stride=2（28→14）。输出 14×14×1024。

**Stage 5 — Conv5_x：** 3 个 bottleneck，`1×1(512)→3×3(512)→1×1(2048)`。第一个 stride=2（14→7）。输出 7×7×2048。

**Classification Head：** Average Pool 7×7 → 1000-d fc → softmax。

## 五、核心创新点

### 创新 1: 残差学习（Residual Learning）

**痛点与动机：** 深层网络优化困难——不是梯度消失，而是"退化"（degradation）。

**为什么已有方法不行：** 添加恒等映射层理论上可以让深层网络至少不差于浅层，但 SGD 无法学到恒等映射。VGG 和 GoogleNet 通过辅助分类器缓解梯度问题，但没解决优化曲面的崎岖性。

**方案细节：** 不学习目标映射 $H(x)$，而是学习残差 $F(x) = H(x) - x$，最终通过 shortcut 得到 $H(x) = F(x) + x$。当恒等映射是最优解时，学习 $F(x)=0$ 比学习 $H(x)=x$ 容易——因为权重初始化为 0 附近，$F(x)$ 天然接近 0。

**为什么有效：** 看反向传播的梯度。损失对输入的梯度：

$$\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial \mathcal{L}}{\partial x'} \cdot \frac{\partial x'}{\partial x} = \frac{\partial \mathcal{L}}{\partial x'} \left(1 + \frac{\partial F}{\partial x}\right)$$

shortcut 连接给梯度提供了一个**直通车**（$\frac{\partial \mathcal{L}}{\partial x'} \cdot 1$），即使 $\frac{\partial F}{\partial x}$ 非常小（深层梯度消失），梯度仍然可以通过 shortcut 无损回传。

**消融证据：** Table 2 显示：34 层 plain（无 shortcut）top-1 错误率 28.54%，34 层 ResNet 7.40——残差连接降低了超过 21 个百分点。18 层 plain 和 ResNet 的差距（27.94 vs 27.88）很小，说明层数越深残差连接越重要。

### 创新 2: 瓶颈设计（Bottleneck Architecture）

**动机：** ResNet-34 用两个 3×3 卷积堆叠，当层数增加到 50+ 时计算量太大。

**方案：** 用 `1×1 → 3×3 → 1×1` 替代 `3×3 → 3×3`。1×1 层先降维再升维，中间的 3×3 在低维空间上计算，FLOPs 大幅降低。

| 模块 | 参数量 | FLOPs |
|------|--------|-------|
| Basic: 3×3,256 → 3×3,256 | 1,179,648 | 1.18M |
| Bottleneck: 1×1,64 → 3×3,64 → 1×1,256 | 69,632 | 0.07M |

**16.9 倍参数减少。** 这让 ResNet-152（60M 参数）成为可能。

## 六、公式详解

### 公式 1: 残差块

$$y = F(x, \{W_i\}) + x$$

其中 $F = W_2 \sigma(W_1 x)$，$\sigma$ 是 ReLU（为简化省略了 BN）。

如果维度不同：$y = F(x, \{W_i\}) + W_s x$，$W_s$ 是 1×1 卷积。

**来源：** 论文 Eq. 1。残差学习的思想源于 HighWay Networks（Srivastava et al., 2015），但 Highway 用了门控（gated shortcut），ResNet 用了无参数恒等映射。

### 公式 2: 梯度流

$$\frac{\partial \mathcal{L}}{\partial x_l} = \frac{\partial \mathcal{L}}{\partial x_L} \left(1 + \frac{\partial}{\partial x_l} \sum_{i=l}^{L-1} F(x_i, W_i)\right)$$

**来源：** 论文 Eq. 4。这是残差网络梯度反传的核心公式——从深层 $x_L$ 到浅层 $x_l$ 的梯度包含两项：$\frac{\partial \mathcal{L}}{\partial x_L}$（直通项）和 $\frac{\partial \mathcal{L}}{\partial x_L} \cdot \frac{\partial}{\partial x_l} \sum F$（经过残差层的项）。直通项保证梯度不会消失。

### 公式 3: 残差网络的反向传播特性

$$\frac{\partial \mathcal{L}}{\partial x_l} = \frac{\partial \mathcal{L}}{\partial x_L} + \frac{\partial \mathcal{L}}{\partial x_L} \left(\sum_{i=l}^{L-1} \frac{\partial F}{\partial x_i}\right)$$

**直觉：** 当 $\frac{\partial \mathcal{L}}{\partial x_L}$ 很小（深层梯度消失），第二项可能接近 0，但第一项 $\frac{\partial \mathcal{L}}{\partial x_L}$ 永远存在——梯度不会完全消失。这是残差连接的核心理论保证。

## 八、代码对照（GitHub: KaimingHe/deep-residual-networks）

官方实现是 Caffe。几个关键细节：

**1. 初始化策略。** 各层卷积权重用 $\sqrt{2/n}$ 的零均值高斯分布初始化——$n$ 是输入通道数×卷积核宽×卷积核高。这个初始化是 kaiming_normal 的前身。

**2. 下采样位置。** ResNet 的下采样发生在每个 stage 的第一个卷积（stride=2），不是在 shortcut 上。shortcut projection 只在维度变化时使用。

**3. BN 在激活之前。** 顺序是 Conv → BN → ReLU。这和后来的 Pre-activation ResNet（BN → ReLU → Conv）不同。

**4. 训练细节。** batch size 256，初始 lr 0.1，当误差停滞时除以 10（手动调整），weight decay 0.0001，momentum 0.9，训练 60 万次迭代。

## 九、位置

### 前驱工作

- **[Srivastava et al., 2015] Highway Networks**：第一个引入门控 shortcut 连接的工作。ResNet 去掉了门控，用了恒等映射（无参数）——更简单、更有效。
- **[Simonyan & Zisserman, 2014] VGG**：证明了深度的重要性（16-19 层），但参数过多且训练困难。
- **[Ioffe & Szegedy, 2015] Batch Normalization**：ResNet 依赖 BN 稳定训练，和残差连接配合发挥最大效果。

### 后续影响

ResNet 是 CV 领域引用量最高的论文之一（10 万+ 引用），它的影响远超分类任务本身：

- **ResNet 的设计成为 backbone 的事实标准**：Faster R-CNN、Mask R-CNN、FPN、RetinaNet、U-Net 等所有主流检测/分割框架都基于 ResNet。
- **残差连接成为深度学习的基础组件**：Transformer 的每个 block 都用了残差连接。如果没有 ResNet，今天的 Transformer 架构（BERT、GPT、ViT）可能不会存在。
- **ResNeXt、ResNeSt、WideResNet** 等直接在 ResNet 基础上改进。
- **ResNet 的可扩展性**：ResNet-152 到 ResNet-1001 都可用，验证了残差连接可以支持千层网络。
- **DenseNet**（Huang et al., 2017）在 ResNet 的基础上把 shortcut 扩展为 dense connection。

## 十、局限

1. **不是最深的残差连接方案。** 后续的 Pre-activation ResNet（He et al., 2016）证明把 BN/ReLU 移到卷积前可以训练更深（1001 层）。ResNet 原始设计的 BN 位置不是最优的。

2. **Projection shortcut 不是必须的。** ResNet 在维度变化时用 1×1 卷积调整维度。后续工作证明 zero-padding 也能工作，且 DenseNet 证明了（论文 Appendix A）。这个设计选择增加了些许实现复杂度。

3. **训练时间长。** ResNet-152 在 8 块 GPU 上训练 2-3 周。后续的 warmup + cosine decay 和更大的 batch size 缩短了训练时间。

4. **没有理论证明为什么残差有效。** 论文给出了梯度分析，但没有严格的理论证明。后续（Veit et al., 2016）解释了 ResNet 是许多浅层网络的 ensemble。

## 十一、小结

ResNet 的核心贡献不是一个复杂的架构，而是一个极其简单的操作——恒等映射 shortcut 连接。它在 ImageNet 上以 VGG-16 不到一半的参数超越了 VGG-16，并让 152 层的网络变得可训练。

但 ResNet 最重要的遗产不是它自身的精度，而是它解决了"深层网络可以更深"这个基本问题：残差连接让梯度可以从最后一层无损地回传到第一层。这个思想被 Transformer 借鉴后，推动了 BERT、GPT、ViT 等一系列深度架构的诞生。没有 ResNet，深度学习可能还在 20 层的瓶颈上徘徊。
