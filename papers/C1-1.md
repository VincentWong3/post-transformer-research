# Faster R-CNN: Towards Real-Time Object Detection with Region Proposal Networks（2015）

**论文：** [arXiv](https://arxiv.org/abs/1506.01497) · Ren et al. · 2015

**RPN（Region Proposal Network）和检测网络共享卷积特征。** Faster R-CNN 把 region proposal 从外部算法（Selective Search）变成了一个可学习的网络。RPN 在特征图上滑动 anchor 框，输出 objectness score + box regression。这是 DETR 之前最广泛使用的检测框架，两阶段检测的事实标准。
