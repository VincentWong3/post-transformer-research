# Tokens-to-Token ViT: Training Vision Transformers from Scratch on ImageNet（2021）

**论文：** [arXiv](https://arxiv.org/abs/2101.11986) · Li Yuan, Yunpeng Chen, Tao Wang et al. · 2021

---

## 一、核心问题

ViT 的 patch token 化（16×16）是一次性完成的，丢失了局部结构信息（边缘、纹理）。T2T 的核心改进：

1. **渐进式 token 化**——先用小窗口（7×7）提取局部信息，逐步合并 tokens
2. **减少序列长度**——从 1024 tokens → 256 tokens → final feature
3. **引入 CNN 归纳偏置**——使用 soft split（overlapping unfolding）保留局部结构

## 二、架构

T2T 模块替代 ViT 的单一 patch embedding：

```
Input: 224×224×3
    │
    ▼
T2T Module:
  Step 1: Soft Split 7×7, stride=3 → 1024 tokens, dim=147
  Step 2: Transformer ×1 → 新的 1024 tokens
  Step 3: Soft Split 3×3, stride=2 → 256 tokens, dim=... 
  Step 4: Transformer ×1 → 256 tokens
  Step 5: Flatten → final tokens
    │
    ▼
  Transformer Encoder ×L（T2T-ViT）
```

T2T-ViT-14 在 ImageNet-1K 从零训练达到 81.5%，超过 DeiT-S（81.2%）。
