# Per-Pixel Classification is Not All You Need for Semantic Segmentation（2021）

**论文：** [arXiv](https://arxiv.org/abs/2107.06278) · Cheng et al. · 2021（MaskFormer）

**核心创新：把分割统一为 mask classification 问题。** 不再是逐像素分类（per-pixel classification），而是：预测一组 mask proposals + 每个 mask 的类别概率。这个统一框架可以处理语义分割、实例分割、全景分割。

---

# Masked-attention Mask Transformer for Universal Image Segmentation（2022）

**论文：** [arXiv](https://arxiv.org/abs/2112.01527) · Cheng et al. · 2022（Mask2Former）

基于 MaskFormer，用 masked attention 让 query 只关注预测的 mask 区域内的特征。通过使用多分辨率特征 + 更少的 query + 更小的 Transformer，Mask2Former 在精度和效率上都超过了 MaskFormer。统一语义/实例/全景分割的 SOTA。

# OneFormer: One Transformer to Rule Universal Image Segmentation（2023）

**论文：** [arXiv](https://arxiv.org/abs/2211.06220) · Jain et al. · 2023

**用一个模型同时处理语义/实例/全景分割，不需要为每个任务单独训练。** 通过 task token 控制输出格式。证明三者之间的差异可以通过任务条件输入统一。
