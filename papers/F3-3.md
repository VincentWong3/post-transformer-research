# DeepNet: Scaling Transformers to 1,000 Layers（2022）

**论文：** [arXiv](https://arxiv.org/abs/2203.00555) · 作者：DeepNet Team (Microsoft Research) · 年份：2022

---

## 一、先搞清楚坑在哪

Transformer 架构的成功已经不需要多说了——从 NLP 到 CV，从 BERT 到 ViT，大家都在拼命把模型越做越深。直观上，层数越多，模型容量越大，表达能力越强。但现实很骨感：当你把 Transformer 堆到几十层以上时，训练就开始不稳定了，梯度爆炸、梯度消失、输出震荡，各种问题接踵而至。

**这里有个关键矛盾**：层数越深，模型越强，但越难训练。这就像盖摩天大楼——地基稍微有点偏差，上面几十层就歪了。Transformer 的残差连接（Residual Connection）和 LayerNorm 虽然能缓解问题，但远远不够。

## 二、现有方法的真正问题

### 2.1 残差连接 + LayerNorm 的局限性

先看看标准的 Pre-LN Transformer block：

```
x_{l+1} = x_l + F(LN(x_l))
```

其中 F 是 attention 或 FFN，LN 是 LayerNorm。这个设计在几十层时还行，但到几百层时就不行了。为什么？

**核心问题**：残差分支的信号会随着层数累积。每经过一层，残差分支的输出 F(LN(x_l)) 会叠加到主分支 x_l 上。当层数达到 1000 层时，主分支的信号会被大量残差信号"稀释"，导致模型退化。

### 2.2 梯度问题

从梯度角度看，反向传播时：

```
∂L/∂x_l = ∂L/∂x_L * Π(1 + ∂F/∂x_k)
```

当层数 L 很大时，这个乘积要么爆炸（如果 ∂F/∂x_k > 0 且很大），要么消失（如果 ∂F/∂x_k < 0 且绝对值大）。虽然残差连接让梯度有了一条"高速公路"（1 的部分），但 ∂F/∂x_k 的累积效应仍然存在。

### 2.3 现有解决方案的不足

| 方法 | 思路 | 局限 |
|------|------|------|
| Pre-LN | LayerNorm 放在残差分支前 | 深层时信号衰减严重 |
| Post-LN | LayerNorm 放在残差分支后 | 训练不稳定，需要 warmup |
| ReZero | 残差分支乘以可学习标量 | 深层时梯度仍然有问题 |
| Fixup | 特殊初始化 + 残差缩放 | 理论上限有限 |

**有意思的是**：这些方法都在"调残差连接的强度"，但没有从根本上解决信号在深层传播时的稳定性问题。

## 三、DeepNet 的核心思路

DeepNet 的核心洞察非常简洁：**不是残差连接有问题，而是残差连接的初始化方式有问题。**

具体来说，DeepNet 提出了一个叫 **DeepNorm** 的初始化 + 归一化方案，让 Transformer 可以稳定地训练到 1000 层，而且不需要任何特殊的训练技巧（如学习率 warmup、梯度裁剪等）。

**一句话总结**：DeepNorm 通过调整残差连接的初始缩放和 LayerNorm 的更新方式，确保信号在深层传播时既不会被放大也不会被衰减。

## 四、网络架构详解

### 4.1 整体架构

DeepNet 的架构和标准 Transformer 几乎一样，唯一的区别是每个 block 中多了一个缩放因子 α：

```
x_{l+1} = LN(α · x_l + F(x_l))
```

其中 α 是一个常数，由层数决定：α = (2N)^{1/2}（N 是总层数）。

**等等，这个公式是不是看起来太简单了？** 确实，核心改动就这么一行。但正是这行改动，解决了深层训练的稳定性问题。

### 4.2 架构图

```
输入序列: [batch, seq_len, d_model]
    │
    ▼
┌─────────────────────────────────────┐
│          Embedding Layer             │
│   Token Embedding + Position Encoding │
│        输出: [B, S, d_model]          │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│         DeepNet Block × N            │
│  (N 可以是 1000 层)                   │
│                                      │
│  ┌─────────────────────────────┐     │
│  │   Multi-Head Self-Attention  │     │
│  │   + DeepNorm                 │     │
│  └─────────────────────────────┘     │
│                                      │
│  ┌─────────────────────────────┐     │
│  │   Feed-Forward Network      │     │
│  │   + DeepNorm                 │     │
│  └─────────────────────────────┘     │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│           Output Layer               │
│   LayerNorm + Linear + Softmax       │
└─────────────────────────────────────┘
    │
    ▼
输出: [batch, seq_len, vocab_size]
```

### 4.3 前向过程详解

**Step 1: Embedding**
- 输入: token IDs [B, S]
- Token Embedding: [B, S, d_model]
- Position Encoding: [B, S, d_model]（使用正弦/余弦编码）
- 输出: x_0 = Embedding + PositionEncoding, shape [B, S, d_model]

**Step 2: DeepNet Block（重复 N 次）**

对于第 l 层（l = 0, 1, ..., N-1）：

**子层 1: Multi-Head Self-Attention**
```
x_l → LN → QKV Projection → Attention → Output Projection → Dropout
     ↓
x_{l+1/2} = DeepNorm(x_l + Attention(LN(x_l)))
```

这里 DeepNorm 的定义：
```
DeepNorm(x) = LN(α · x_l + F(x_l))
```
其中 α = (2N)^{1/2}，F 是 attention 或 FFN。

**子层 2: Feed-Forward Network**
```
x_{l+1/2} → LN → Linear → ReLU → Linear → Dropout
     ↓
x_{l+1} = DeepNorm(x_{l+1/2} + FFN(LN(x_{l+1/2})))
```

**Step 3: Output**
- 经过 N 个 block 后，得到 x_N [B, S, d_model]
- 应用最后的 LayerNorm
- 通过 Linear 层映射到 vocab_size
- Softmax 得到概率分布

### 4.4 为什么这个设计能 work？

**直觉**：在标准 Transformer 中，残差连接是"不加缩放"的叠加。当层数很多时，信号会指数级增长或衰减。DeepNorm 通过 α 这个缩放因子，让信号在每一层都保持在一个稳定的范围内。

具体来说：
- α = (2N)^{1/2} 确保了信号在 N 层后的方差为 O(1)
- LayerNorm 进一步归一化，确保每层的输出都在合理范围
- 两者结合，让梯度也能稳定传播

## 五、核心创新点

### 创新点 1: DeepNorm 初始化

#### (a) 痛点与动机

**问题**：标准 Transformer 在深层时，残差连接会导致信号方差爆炸。

**为什么现有方法失败**：
- [Vaswani et al., 2017] 使用 Post-LN，需要 warmup 才能稳定训练
- [Xiong et al., 2020] 提出 Pre-LN，但深层时信号衰减严重
- [Bachlechner et al., 2020] 的 ReZero 在 100 层以上就不稳定了

**具体观察**：当层数 N 很大时，残差分支的输出 F(x) 的方差会累积。假设每层 F(x) 的方差为 σ²，N 层后主分支的方差约为 Nσ²。这导致模型输出要么爆炸（如果 σ² > 0），要么消失（如果 σ² < 0）。

#### (b) 方案细节

DeepNorm 的定义：

```
DeepNorm(x) = LayerNorm(α · x + F(x))
```

其中：
- α = (2N)^{1/2}，N 是总层数
- F(x) 是 attention 或 FFN
- LayerNorm 是标准的层归一化

**关键**：α 不是可学习的参数，而是由层数决定的常数。

#### (c) 为什么有效

**数学直觉**：
1. 假设每层 F(x) 的方差为 σ²_F
2. 经过 α 缩放后，残差项的方差为 α² · σ²_F
3. 经过 N 层后，主分支的方差约为 N · α² · σ²_F
4. 代入 α² = 2N，得到方差 ≈ 2N² · σ²_F

等等，这不是更大了吗？**关键在这里**：LayerNorm 会在每层之后重新归一化，所以实际传播的是归一化后的信号。LayerNorm 确保每层的输出方差为 1，从而打破了方差累积。

**更准确的分析**：
1. 输入 x_l 的方差为 1（经过 LayerNorm）
2. F(x_l) 的方差为 σ²_F（取决于初始化）
3. α · x_l + F(x_l) 的方差为 α² + σ²_F
4. 经过 LayerNorm 后，方差变回 1

所以实际上，DeepNorm 通过 LayerNorm 的"重置"机制，防止了方差的累积。

#### (d) 与 Related Work 的关系

DeepNorm 与以下工作有直接关系：

- **[ReZero, Bachlechner et al., 2020]**：提出用可学习标量缩放残差分支。DeepNorm 改用固定的 α，避免了学习过程中的不稳定。
- **[Fixup, Zhang et al., 2019]**：通过特殊初始化 + 残差缩放来稳定训练。DeepNorm 更简洁，只改了一行代码。
- **[T-Fixup, Huang et al., 2020]**：针对 Transformer 的 Fixup 变体。DeepNorm 不需要修改初始化方式。

**如果不用 DeepNorm 会怎样？** 用 ReZero 的话，1000 层 Transformer 的训练 loss 会发散；用 Pre-LN 的话，模型会退化到几乎无法学习。

#### (e) 消融实验证据

从论文 Table 2 可以看出：

| 方法 | 100 层 | 500 层 | 1000 层 |
|------|--------|--------|---------|
| Pre-LN | ✓ | ✗（发散） | ✗ |
| Post-LN | ✗（需要 warmup） | ✗ | ✗ |
| ReZero | ✓ | ✗（不稳定） | ✗ |
| DeepNorm | ✓ | ✓ | ✓ |

**有意思的是**：DeepNorm 不仅让 1000 层训练可行，而且训练曲线比浅层模型更平滑。

### 创新点 2: 理论稳定性保证

#### (a) 痛点与动机

**问题**：现有的深层 Transformer 训练方法缺乏理论保证，大家都是在"试"。

**为什么现有方法失败**：
- 经验方法（如 warmup）没有理论支撑
- 启发式方法（如梯度裁剪）会改变优化方向

#### (b) 方案细节

论文提供了严格的理论分析，证明 DeepNorm 可以保证：

1. **前向传播稳定性**：任意两层的输出差有界
2. **反向传播稳定性**：梯度不会爆炸或消失
3. **参数更新稳定性**：每层的参数更新量有界

具体来说，论文证明了：

**Theorem 1**：对于任意输入 x，经过 N 层 DeepNet 后，输出 f(x) 满足：
```
||f(x)|| ≤ C · ||x||
```
其中 C 是与 N 无关的常数。

**Theorem 2**：梯度 ∂L/∂x_l 的范数有界，不会随着层数增长。

#### (c) 为什么有效

**直觉**：DeepNorm 的 α 缩放 + LayerNorm 的组合，实际上构建了一个"收缩映射"（contraction mapping）。每一层都在压缩信号空间，确保信号不会发散。

数学上，这类似于动力系统中的 Lyapunov 稳定性——系统的状态轨迹被约束在一个有界区域内。

#### (d) 与 Related Work 的关系

- **[ResNet, He et al., 2016]**：残差连接的理论分析主要关注梯度传播，没有考虑前向稳定性
- **[Mean Field Theory, Poole et al., 2016]**：用统计物理方法分析深层网络，但主要针对全连接网络
- **[Neural ODE, Chen et al., 2018]**：把残差网络看作 ODE 离散化，但需要连续深度假设

DeepNorm 的理论分析填补了 Transformer 深层训练理论保证的空白。

#### (e) 消融实验证据

论文 Figure 3 展示了不同深度下的训练曲线：
- 100 层：收敛速度与浅层模型相当
- 500 层：收敛略慢，但最终性能更好
- 1000 层：仍然能收敛，只是需要更多步数

**关键发现**：即使 1000 层，训练 loss 也没有出现发散或震荡。

## 六、公式详解

### 公式 1: DeepNorm 定义

```
DeepNorm(x) = LayerNorm(α · x + F(x))
```

#### (a) 符号定义
- x: 当前层的输入，shape [B, S, d_model]
- α: 缩放因子，常数 α = (2N)^{1/2}，N 是总层数
- F(x): 子层函数（attention 或 FFN）
- LayerNorm: 标准层归一化

#### (b) 公式来源

这个公式是论文原创的，灵感来自 ReZero 和 Fixup。与 ReZero 的区别在于：
- ReZero: x + α · F(x)，α 是可学习的
- DeepNorm: α · x + F(x)，α 是固定的

**为什么反过来？** 因为缩放主分支比缩放残差分支更稳定。当层数很多时，主分支需要被"压制"（乘以小于 1 的因子），而不是放大残差分支。

#### (c) 推导过程

假设每层 F(x) 的方差为 σ²_F，x 的方差为 σ²_x。

**标准残差连接**：
```
y = x + F(x)
Var(y) = σ²_x + σ²_F + 2Cov(x, F(x))
```
如果 x 和 F(x) 不相关，Var(y) ≈ σ²_x + σ²_F。经过 N 层后，方差 ≈ Nσ²_F（假设 σ²_x ≈ 1）。

**DeepNorm**：
```
y = α · x + F(x)
Var(y) = α² · σ²_x + σ²_F
```
代入 α² = 2N，σ²_x ≈ 1：
```
Var(y) ≈ 2N + σ²_F
```
经过 LayerNorm 后，方差变回 1。

**关键**：LayerNorm 的"重置"作用打破了方差累积。所以实际传播的不是原始方差，而是归一化后的信号。

#### (d) 直觉理解

**具体例子**：假设 N=1000，d_model=1024。

**第一项**：α · x，α = (2×1000)^{1/2} ≈ 44.72
- 输入 x 的每个元素 ~ N(0, 1/d_model)
- 缩放后，每个元素 ~ N(0, α²/d_model) = N(0, 2000/1024) ≈ N(0, 1.95)

**第二项**：F(x)，attention 输出的每个元素 ~ N(0, 1/d_model)（假设 Xavier 初始化）

**第三项**：LayerNorm 将总和归一化回 N(0, 1/d_model)

**直观理解**：α 很大（~45），但 F(x) 很小（~1/√1024 ≈ 0.03）。所以主分支占主导，残差分支只是微调。这类似于"预训练 + 微调"的思路——主分支保持主要信号，残差分支做精细调整。

#### (e) 边界情况

**当 N=1 时**：α = √2 ≈ 1.414，与标准残差连接（α=1）差别不大。

**当 N 很大时**：α 很大，主分支被放大。但 LayerNorm 会重新归一化，所以实际效果是"主分支被压制，残差分支被放大"。

**极端情况 N→∞**：α → ∞，理论上主分支会占绝对主导。但实际中，LayerNorm 会确保残差分支仍然有贡献。

### 公式 2: 梯度传播

```
∂L/∂x_l = ∂L/∂x_N · Π(α + ∂F/∂x_k) / ||α + ∂F/∂x_k||
```

#### (a) 符号定义
- ∂L/∂x_l: 损失对第 l 层输入的梯度
- ∂L/∂x_N: 损失对最后一层输入的梯度
- α: 缩放因子，α = (2N)^{1/2}
- ∂F/∂x_k: 第 k 层子层函数的 Jacobian

#### (b) 公式来源

这是从 DeepNorm 的前向公式推导出来的。标准残差连接的梯度传播是：
```
∂L/∂x_l = ∂L/∂x_N · Π(1 + ∂F/∂x_k)
```

DeepNorm 的梯度传播引入了 α 和 LayerNorm 的归一化项。

#### (c) 推导过程

**标准残差连接**：
```
x_{l+1} = x_l + F(x_l)
∂x_{l+1}/∂x_l = 1 + ∂F/∂x_l
∂L/∂x_l = ∂L/∂x_N · Π(1 + ∂F/∂x_k)
```

**DeepNorm**：
```
x_{l+1} = LN(α · x_l + F(x_l))
∂x_{l+1}/∂x_l = (α + ∂F/∂x_l) / ||α + ∂F/∂x_l||
```
这里 LayerNorm 的梯度引入了归一化项。

所以：
```
∂L/∂x_l = ∂L/∂x_N · Π(α + ∂F/∂x_k) / ||α + ∂F/∂x_k||
```

#### (d) 直觉理解

**具体例子**：假设 N=1000，α=44.72，∂F/∂x_k 的谱范数 ≈ 1。

**标准残差连接**：
```
Π(1 + ∂F/∂x_k) ≈ (1 + 1)^{1000} = 2^{1000}
```
这显然会爆炸。

**DeepNorm**：
```
Π(α + ∂F/∂x_k) / ||α + ∂F/∂x_k|| ≈ (44.72/44.72)^{1000} = 1
```
因为 α 远大于 ∂F/∂x_k，所以每个因子的范数 ≈ 1。

**直观理解**：α 的引入让每个梯度因子的范数都接近 1，从而防止了梯度爆炸或消失。

#### (e) 边界情况

**当 α 远大于 ∂F/∂x_k 时**：每个因子 ≈ 1，梯度稳定传播。

**当 α 与 ∂F/∂x_k 相当时**：梯度可能不稳定，但实际中 α 远大于 ∂F/∂x_k。

**当 ∂F/∂x_k 很大时**：α 仍然占主导，梯度仍然稳定。

### 9.1 前驱工作

2. **[ReZero, Bachlechner et al., 2020]**：提出用可学习标量缩放残差分支。DeepNet 借鉴了"缩放残差"的思路，但改用固定缩放。

3. **[Fixup, Zhang et al., 2019]**：通过特殊初始化 + 残差缩放来稳定训练。DeepNet 更简洁，只改了一行代码。

4. **[T-Fixup, Huang et al., 2020]**：针对 Transformer 的 Fixup 变体。DeepNet 不需要修改初始化方式。

5. **[Pre-LN, Xiong et al., 2020]**：证明 Pre-LN 比 Post-LN 更稳定。DeepNet 使用 Pre-LN 风格（LayerNorm 在残差分支前）。

### 9.2 同期竞品

1. **[NormFormer, Shleifer et al., 2021]**：在 attention 和 FFN 之间添加额外的 LayerNorm。与 DeepNet 思路不同，NormFormer 更关注每层的归一化。

2. **[SandwichNorm, Anonymous, 2021]**：在 attention 前后都加 LayerNorm。与 DeepNet 的"缩放 + 归一化"思路不同。

3. **[ScaleNorm, Nguyen et al., 2021]**：用可学习缩放替代 LayerNorm。与 DeepNet 的固定缩放形成对比。

4. **[BranchNorm, Anonymous, 2022]**：在残差分支上使用可学习缩放。与 DeepNet 的固定缩放 + LayerNorm 不同。

## 十、局限

### 局限 1: 计算成本

**问题**：1000 层 Transformer 的计算成本极高。即使训练稳定，推理速度也慢得无法接受。

**什么时候重要**：在实际部署中，100 层和 1000 层的推理速度差 10 倍，但性能提升可能只有 0.5 BLEU。对于工业应用来说，性价比不高。

**论文是否承认**：论文没有直接讨论推理成本问题，只关注了训练稳定性。

### 局限 2: 性能收益递减

**问题**：从 500 层到 1000 层，参数量翻倍，但 BLEU 只提升了 0.2。性能收益递减明显。

**什么时候重要**：当资源有限时，增加层数不如增加宽度或数据量。

**论文是否承认**：论文提到"性能随层数增加而提升"，但没有讨论收益递减问题。

## 十一、小结

**问题**：Transformer 在深层时训练不稳定，现有方法（Pre-LN、ReZero、Fixup）在 100 层以上就失效了。

**解决方案**：DeepNet 提出了 DeepNorm，通过 α = (2N)^{1/2} 缩放主分支 + LayerNorm 归一化，让信号在深层传播时保持稳定。

**为什么有效**：α 缩放 + LayerNorm 的组合构建了一个"收缩映射"，防止了信号方差累积和梯度爆炸/消失。理论分析证明了前向和反向传播的稳定性。

**更广泛的影响**：DeepNet 证明了 1000 层 Transformer 是可行的，为超深层模型的研究打开了大门。虽然实际部署中计算成本限制了其应用，但 DeepNorm 的缩放技巧已经被多个后续工作采纳（PaLM、LLaMA、DeepSpeed 等）。

**有意思的是**：最有效的改动往往是最简单的。DeepNorm 只改了一行代码（添加 α 缩放），就解决了困扰领域多年的深层训练问题。这提醒我们，有时候复杂的问题不需要复杂的解决方案。
