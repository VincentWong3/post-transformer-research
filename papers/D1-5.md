# ALIGN: Large-scale Image and Noisy-Text Embedding（2021）

**论文：** [arXiv](https://arxiv.org/abs/2102.05918) · Jia et al. · 2021

类似 CLIP 的图文对比学习，但数据规模更大（1.8B 图文对，来自 alt-text）。使用双线性 attention pooling 替代 CLIP 的 [CLS] token，在图文检索（Flickr30K、COCO Captions）上表现出色。

# Sigmoid Loss for Language Image Pre-Training（2023）

**论文：** [arXiv](https://arxiv.org/abs/2303.15343) · Zhai et al. · 2023（SigLIP）

用 sigmoid loss 替代 CLIP 的 softmax contrastive loss。每对正负样本独立计算损失，不再需要全局归一化。好处是 batch size 可以更小，训练更稳定。SigLIP 用 256 batch size 达到了 CLIP 用 32k batch size 的效果。
